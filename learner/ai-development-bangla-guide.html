AI (কৃত্রিম বুদ্ধিমত্তা) আসলে কী?
Artificial Intelligence — মূল ধারণা থেকে শুরু

Artificial Intelligence (AI) হলো কম্পিউটার সিস্টেমকে এমনভাবে তৈরি করা যেন সে মানুষের মতো চিন্তা করতে, শিখতে এবং সমস্যা সমাধান করতে পারে। সহজ ভাষায় — মেশিনকে "বুদ্ধিমান" করে তোলার প্রযুক্তি।

আজকের AI মূলত Large Language Models (LLM) এর উপর ভিত্তি করে — যেগুলো কোটি কোটি টেক্সট পড়ে শিখেছে এবং এখন যেকোনো প্রশ্নের উত্তর দিতে, কোড লিখতে, ছবি বিশ্লেষণ করতে পারে।

🤖 Generative AI

টেক্সট, ছবি, ভিডিও, কোড তৈরি করে। ChatGPT, Claude, Gemini, Midjourney এই ধরনের।

🔍 Predictive AI

ডেটা দেখে ভবিষ্যৎ অনুমান করে। স্প্যাম ফিল্টার, রেকমেন্ডেশন সিস্টেম এই ধরনের।

👁️ Computer Vision

ছবি ও ভিডিও বোঝে। মুখ চেনা, বস্তু সনাক্তকরণ, OCR এই ধরনের।

🗣️ Speech AI

কথা শুনে বোঝে এবং কথা বলতে পারে। Siri, Google Assistant, Whisper এই ধরনের।

🌐 Translation AI

এক ভাষা থেকে আরেক ভাষায় অনুবাদ করে। Google Translate, DeepL এই ধরনের।

🎯 Recommendation AI

ব্যবহারকারীর পছন্দ অনুযায়ী সাজেশন দেয়। Netflix, YouTube, Amazon এই ধরনের।

💡
ওয়েব ডেভেলপার হিসেবে আপনাকে AI "বানাতে" হবে না! বেশিরভাগ ক্ষেত্রে আপনি শুধু OpenAI, Claude, Gemini এর মতো ready-made AI-এর API ব্যবহার করবেন। এটাই সহজ, দ্রুত এবং শক্তিশালী।
📊
AI কীভাবে কাজ করে?
Machine Learning, Deep Learning, LLM — সহজ বাংলায়

১. Traditional Programming (পুরানো পদ্ধতি)

মানুষ নিয়ম লেখে → কম্পিউটার সেই নিয়ম মেনে চলে। Input দিলে নির্দিষ্ট Output আসে। কিন্তু নতুন পরিস্থিতিতে কাজ করে না।

২. Machine Learning (শেখার পদ্ধতি)

মানুষ data দেয় → কম্পিউটার নিজেই প্যাটার্ন খোঁজে → নতুন data-তেও কাজ করে। যত বেশি data, তত ভালো শেখা।

৩. Deep Learning (গভীর শেখা)

মানুষের মস্তিষ্কের মতো "Neural Network" ব্যবহার করে। লক্ষ লক্ষ প্যারামিটার দিয়ে জটিল প্যাটার্ন ধরতে পারে।

৪. Large Language Models — LLM (আধুনিক AI)

ইন্টারনেটের কোটি কোটি টেক্সট দিয়ে train করা। "Transformer" আর্কিটেকচার ব্যবহার করে। ChatGPT, Claude, Gemini এই ধরনের।

৫. Multimodal AI (২০২৪-২০২৫)

শুধু টেক্সট নয়, ছবি + টেক্সট + অডিও একসাথে বোঝে। GPT-4o, Claude 3.5, Gemini Ultra এই ধরনের।

🔑 API কী এবং কেন গুরুত্বপূর্ণ?

API (Application Programming Interface) হলো দুটি সফটওয়্যারের মধ্যে যোগাযোগের মাধ্যম। AI কোম্পানিগুলো API দেয় যাতে আপনি তাদের AI-কে আপনার অ্যাপে ব্যবহার করতে পারেন।

সহজ উদাহরণ: আপনার Laravel অ্যাপ OpenAI-এর API-কে বলবে "এই প্রশ্নের উত্তর দাও" → OpenAI উত্তর দিবে → আপনার অ্যাপ সেটা ব্যবহারকারীকে দেখাবে।

📡 AI API কীভাবে কাজ করে — সরল উদাহরণ
Concept 
/*
 * AI API এর মূল ধারণা
 * আপনার App → HTTP Request → AI Server → Response → আপনার App
 */

// ১. আপনি একটি Request পাঠান
POST https://api.openai.com/v1/chat/completions
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

{
  "model": "gpt-4o",
  "messages": [
    {
      "role": "user",
      "content": "বাংলাদেশের রাজধানী কী?"
    }
  ]
}

// ২. AI একটি Response দেয়
{
  "choices": [{
    "message": {
      "role": "assistant",
      "content": "বাংলাদেশের রাজধানী ঢাকা।"
    }
  }],
  "usage": {
    "prompt_tokens": 15,
    "completion_tokens": 10,
    "total_tokens": 25    // এই token-এর জন্য পয়সা লাগে
  }
}

📊 Token হিসাব — আসলে কত টাকা লাগে?

Token হলো AI-এর measurement unit। মোটামুটি ১ token = ৪ character বা ৩/৪ শব্দ। "Hello, how are you?" ≈ ৫ token।

ব্যবহারমডেলপ্রতি request tokenমাসে cost (১০০০ req)
সাধারণ ChatbotGPT-4o Mini~500 tokens~$0.04 (মাত্র ৪ টাকা!)
Content GenerationClaude Haiku~2000 tokens~$0.25 (২৫ টাকা)
Code AssistantGPT-4o~3000 tokens~$4.50 (৪৫০ টাকা)
Document AnalysisGemini Pro~10000 tokens~$3.50 (৩৫০ টাকা)

বেশিরভাগ ছোট-মাঝারি প্রজেক্টে মাসে $5-$20 এর বেশি লাগে না। শুরুতে ফ্রি tier দিয়ে শুরু করুন।

Frontend থেকে Backend — AI integration এর সব পদ্ধতি
১

Direct API Call (সরাসরি AI API call)

আপনার backend থেকে সরাসরি OpenAI/Claude/Gemini API-তে HTTP request পাঠান। সবচেয়ে সাধারণ পদ্ধতি। Laravel-এ HTTP Facade বা SDK ব্যবহার করুন।

২

SDK / Package (লাইব্রেরি ব্যবহার)

প্রতিটি AI কোম্পানির official PHP/Laravel SDK আছে। এগুলো ব্যবহার করলে কম কোড লিখতে হয়, error handling ভালো হয়।

৩

Streaming Response (রিয়েলটাইম আউটপুট)

ChatGPT এর মতো টাইপ করতে করতে দেখানো। Server-Sent Events (SSE) বা WebSocket ব্যবহার করে। ব্যবহারকারীর অভিজ্ঞতা অনেক ভালো হয়।

    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-CSRF-TOKEN': document.querySelector('meta[name="csrf-token"]').content

// Streaming response (ChatGPT-এর মতো)
async function streamAI(message, outputElement) {
  const response = await fetch('/api/ai/stream', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ message })
  });

  const reader = response.body.getReader();
  const decoder = new TextDecoder();

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    outputElement.textContent += decoder.decode(value); // টাইপ করতে করতে দেখায়
  }
}

namespace App\Http\Controllers;

use Illuminate\Http\Request;
use OpenAI\Laravel\Facades\OpenAI;

class AIController extends Controller
{
    /**
     * সাধারণ Chat — একটি প্রশ্নের উত্তর দেওয়া
     */
    public function chat(Request $request)
    {
        $request->validate([
            'message' => 'required|string|max:2000'
        ]);

        try {
            $response = OpenAI::chat()->create([
                'model'   => config('openai.request.model', 'gpt-4o-mini'),
                'messages' => [
                    [
                        'role'    => 'system',
                        'content' => 'তুমি একজন সহায়ক AI assistant। বাংলায় উত্তর দাও।'
                    ],
                    [
                        'role'    => 'user',
                        'content' => $request->message
                    ]
                ],
                'max_tokens'  => 1000,
                'temperature' => 0.7, // 0 = সঠিক, 1 = creative
            ]);

            return response()->json([
                'success' => true,
                'reply'   => $response->choices[0]->message->content,
                'tokens'  => $response->usage->totalTokens,
            ]);

        } catch (\Exception $e) {
            return response()->json([
                'success' => false,
                'error'   => 'AI সার্ভিস সাময়িক সমস্যায়। আবার চেষ্টা করুন।'
            ], 500);
        }
    }

    /**
     * Conversation History সহ Chat (Multi-turn)
     * ব্যবহারকারী আগের কথা মনে রাখে
     */
    public function conversationChat(Request $request)
    {
        $messages = $request->input('history', []);

        // System prompt সবার আগে
        $systemMessage = [[
            'role'    => 'system',
            'content' => 'তুমি একটি e-commerce ওয়েবসাইটের customer support AI।
                          বাংলা ও ইংরেজি দুটোতেই উত্তর দিতে পারো।
                          সবসময় বিনয়ী থাকো।'
        ]];

        $allMessages = array_merge($systemMessage, $messages);

        $response = OpenAI::chat()->create([
            'model'    => 'gpt-4o-mini',
            'messages' => $allMessages,
        ]);

        return response()->json([
            'reply' => $response->choices[0]->message->content
        ]);
    }

    /**
     * ছবি বিশ্লেষণ — Vision API
     * ছবি দেখে বর্ণনা দেওয়া
     */
    public function analyzeImage(Request $request)
    {
        $imageUrl = $request->image_url;

        $response = OpenAI::chat()->create([
            'model'    => 'gpt-4o', // Vision-এর জন্য gpt-4o
            'messages' => [[
                'role'    => 'user',
                'content' => [
                    ['type' => 'text', 'text' => 'এই ছবিটি বাংলায় বর্ণনা করো।'],
                    ['type' => 'image_url', 'image_url' => ['url' => $imageUrl]]
                ]
            ]]
        ]);

        return response()->json([
            'description' => $response->choices[0]->message->content
        ]);
    }

    /**
     * Content Generation — Blog Post / Product Description
     */
    public function generateContent(Request $request)
    {
        $topic = $request->topic;
        $type  = $request->type; // 'blog', 'product', 'email'

        $prompts = [
            'blog'    => "'$topic' বিষয়ে ৫০০ শব্দের একটি SEO-friendly বাংলা ব্লগ পোস্ট লেখো।",
            'product' => "'$topic' পণ্যের একটি আকর্ষণীয় বাংলা বিবরণ লেখো যা বিক্রি বাড়াবে।",
            'email'   => "'$topic' বিষয়ে একটি প্রফেশনাল বাংলা ইমেইল লেখো।",
        ];

        $response = OpenAI::chat()->create([
            'model'       => 'gpt-4o-mini',
            'messages'    => [['role' => 'user', 'content' => $prompts[$type]]],
            'max_tokens'  => 1500,
        ]);

        return response()->json([
            'content' => $response->choices[0]->message->content
        ]);
    }
}

PHP / Laravel 


namespace App\Services;

use Illuminate\Support\Facades\Http;

class GeminiService
{
    private string $apiKey;
    private string $model;
    private string $baseUrl = 'https://generativelanguage.googleapis.com/v1beta/models';

    public function __construct()
    {
        $this->apiKey = config('services.gemini.api_key');
        $this->model  = config('services.gemini.model', 'gemini-1.5-flash');
    }

    /**
     * টেক্সট Generate করা
     */
    public function generate(string $prompt): string
    {
        $url = "{$this->baseUrl}/{$this->model}:generateContent?key={$this->apiKey}";

        $response = Http::post($url, [
            'contents' => [[
                'parts' => [['text' => $prompt]]
            ]],
            'generationConfig' => [
                'temperature'     => 0.7,
                'maxOutputTokens' => 1000,
            ]
        ]);

        return $response->json('candidates.0.content.parts.0.text');
    }

    /**
     * Conversation Chat (history সহ)
     */
    public function chatWithHistory(array $history, string $newMessage): string
    {
        $url = "{$this->baseUrl}/{$this->model}:generateContent?key={$this->apiKey}";

        // History format করা
        $contents = collect($history)->map(fn($msg) => [
            'role'  => $msg['role'] === 'assistant' ? 'model' : 'user',
            'parts' => [['text' => $msg['content']]]
        ])->push([
            'role'  => 'user',
            'parts' => [['text' => $newMessage]]
        ])->toArray();

        $response = Http::post($url, ['contents' => $contents]);

        return $response->json('candidates.0.content.parts.0.text');
    }
}

namespace App\Services;

use Illuminate\Support\Facades\Http;

class OllamaService
{
    private string $baseUrl;
    private string $model;

    public function __construct()
    {
        $this->baseUrl = env('OLLAMA_BASE_URL', 'http://localhost:11434');
        $this->model   = env('OLLAMA_MODEL', 'llama3.2');
    }

    /**
     * Simple Chat — API cost শূন্য!
     */
    public function chat(string $message, string $systemPrompt = ''): string
    {
        $messages = [];

        if ($systemPrompt) {
            $messages[] = ['role' => 'system', 'content' => $systemPrompt];
        }

        $messages[] = ['role' => 'user', 'content' => $message];

        $response = Http::timeout(120)->post("{$this->baseUrl}/api/chat", [
            'model'    => $this->model,
            'messages' => $messages,
            'stream'   => false,
        ]);

        return $response->json('message.content');
    }

    /**
     * যেসব মডেল ইন্সটল আছে তার তালিকা
     */
    public function listModels(): array
    {
        $response = Http::get("{$this->baseUrl}/api/tags");
        return $response->json('models', []);
    }
}

namespace App\Http\Controllers;

use Illuminate\Http\Request;
use OpenAI\Laravel\Facades\OpenAI;
use App\Models\AiConversation;
use Illuminate\Support\Str;

class ChatbotController extends Controller
{
    // আপনার ব্যবসার তথ্য AI-কে দিন
    private string $systemPrompt = <<;

    public function chat(Request $request)
    {
        $request->validate([
            'message'    => 'required|string|max:500',
            'session_id' => 'nullable|string',
        ]);

        $sessionId = $request->session_id ?? Str::uuid();
        $userId    = auth()->id();

        // ─── আগের কথোপকথন লোড করুন ───
        $history = AiConversation::
            where('session_id', $sessionId)
            ->orderBy('created_at')
            ->take(10) // শেষ ১০টি message
            ->get()
            ->map(fn($m) => [
                'role'    => $m->role,
                'content' => $m->content
            ])->toArray();

        // ─── OpenAI Call ───
        $response = OpenAI::chat()->create([
            'model'    => 'gpt-4o-mini',
            'messages' => array_merge(
                [['role' => 'system', 'content' => $this->systemPrompt]],
                $history,
                [['role' => 'user', 'content' => $request->message]]
            ),
            'max_tokens'  => 300,
            'temperature' => 0.5,
        ]);

        $aiReply = $response->choices[0]->message->content;

        // ─── Database-এ সংরক্ষণ ───
        AiConversation::insert([
            ['user_id' => $userId, 'session_id' => $sessionId,
             'role' => 'user', 'content' => $request->message,
             'tokens_used' => $response->usage->promptTokens,
             'created_at' => now(), 'updated_at' => now()],
            ['user_id' => $userId, 'session_id' => $sessionId,
             'role' => 'assistant', 'content' => $aiReply,
             'tokens_used' => $response->usage->completionTokens,
             'created_at' => now(), 'updated_at' => now()],
        ]);

        return response()->json([
            'reply'      => $aiReply,
            'session_id' => $sessionId,
            'tokens'     => $response->usage->totalTokens,
        ]);
    }
}

RAG — আপনার Document দিয়ে AI Chatbot (Laravel)

# Package ইন্সটল
composer require openai-php/laravel
composer require pgvector/pgvector  # PostgreSQL vector extension

// Document Embedding তৈরি করা
public function embedDocument(string $text): array
{
    $response = OpenAI::embeddings()->create([
        'model' => 'text-embedding-3-small',
        'input' => $text,
    ]);

    return $response->embeddings[0]->embedding;
    // এই 1536-মাত্রার vector database-এ সংরক্ষণ করুন
}

// প্রশ্নের উত্তর খোঁজা
public function answerWithRAG(string $question): string
{
    // ১. প্রশ্নের embedding তৈরি
    $questionEmbedding = $this->embedDocument($question);

    // ২. Similar document chunks খোঁজা (pgvector)
    $relevantChunks = DB::select("
        SELECT content FROM document_chunks
        ORDER BY embedding <-> ?::vector
        LIMIT 5
    ", [json_encode($questionEmbedding)]);

    $context = collect($relevantChunks)->pluck('content')->join("\n\n");

    // ৩. Context সহ AI-কে প্রশ্ন করা
    $response = OpenAI::chat()->create([
        'model'    => 'gpt-4o-mini',
        'messages' => [[
            'role'    => 'system',
            'content' => "শুধু নিচের তথ্য ব্যবহার করে বাংলায় উত্তর দাও:\n\n$context"
        ], [
            'role'    => 'user',
            'content' => $question
        ]]
    ]);

    return $response->choices[0]->message->content;
}
