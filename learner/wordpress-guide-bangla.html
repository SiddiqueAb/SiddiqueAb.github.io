WordPress পরিচিতি ও ইনস্টলেশন

💡
WordPress কী?
WordPress হলো বিশ্বের সবচেয়ে জনপ্রিয় CMS (Content Management System)। ইন্টারনেটের প্রায় ৪৩% ওয়েবসাইট WordPress দিয়ে তৈরি। এটি PHP ও MySQL দিয়ে তৈরি একটি ওপেন-সোর্স প্ল্যাটফর্ম।
🖥️
লোকাল ইনস্টলেশন (XAMPP)
প্রথমে XAMPP ডাউনলোড করুন। Apache ও MySQL চালু করুন। htdocs ফোল্ডারে WordPress কপি করুন। Browser-এ localhost/wordpress খুলুন।
☁️
হোস্টিংয়ে ইনস্টলেশন
cPanel থেকে Softaculous ব্যবহার করুন। ১ ক্লিকে WordPress ইনস্টল হবে। অথবা FTP দিয়ে ম্যানুয়ালি ইনস্টল করুন।
🗄️
ডেটাবেস সেটআপ
phpMyAdmin খুলুন। নতুন Database তৈরি করুন। wp-config.php ফাইলে DB_NAME, DB_USER, DB_PASSWORD সেট করুন।

// ডেটাবেস কনফিগারেশন define( 'DB_NAME', 'আপনার_ডেটাবেস_নাম' ); define( 'DB_USER', 'root' ); define( 'DB_PASSWORD', 'আপনার_পাসওয়ার্ড' ); define( 'DB_HOST', 'localhost' ); define( 'DB_CHARSET', 'utf8mb4' ); // সিকিউরিটি কী (wordpress.org/secret-key থেকে নিন) define( 'AUTH_KEY', 'unique-phrase-here' ); define( 'SECURE_AUTH_KEY', 'unique-phrase-here' ); // ডিবাগ মোড (ডেভেলপমেন্টের সময়) define( 'WP_DEBUG', true ); define( 'WP_DEBUG_LOG', true );

সবসময় Child Theme ব্যবহার করুন!

functions.php (Child Theme)
<?php // প্যারেন্ট ও চাইল্ড থিমের স্টাইল লোড করা add_action( 'wp_enqueue_scripts', 'child_theme_styles' ); function child_theme_styles() { // প্যারেন্ট থিম স্টাইল wp_enqueue_style( 'parent-style', get_template_directory_uri() .
