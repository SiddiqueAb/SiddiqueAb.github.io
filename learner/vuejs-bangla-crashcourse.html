01
পরিচিতি
Vue.js কী এবং কেন?
Beginner

Vue.js হলো একটি progressive JavaScript framework যা ব্যবহারকারী ইন্টারফেস (UI) তৈরির জন্য ব্যবহৃত হয়। এটি শেখা সহজ, কিন্তু জটিল অ্যাপ্লিকেশনও বানানো যায়।

🎯
Progressive
একটু একটু করে শেখা যায়। ছোট অংশে বা পুরো SPA হিসেবে ব্যবহার করা যায়।
⚡
Reactive
Data পরিবর্তন হলে UI স্বয়ংক্রিয়ভাবে আপডেট হয়। DOM নিজে ম্যানেজ করতে হয় না।
🧩
Component-based
পুরো UI ছোট ছোট কম্পোনেন্টে ভাগ করা যায়। Reusable ও maintainable।
💡

Vue vs React vs Angular: React শুধু UI library, Angular পুরো framework (জটিল), Vue মাঝামাঝি — শেখা সহজ কিন্তু পুরো framework-এর সুবিধা আছে।

03
Core
Reactivity — ref() ও reactive()
Beginner

Vue-এর সবচেয়ে শক্তিশালী বৈশিষ্ট্য হলো Reactivity। Data পরিবর্তন হলে DOM স্বয়ংক্রিয়ভাবে আপডেট হয়। এর জন্য দুটি পদ্ধতি আছে: ref() এবং reactive()।

🔵
ref()

যেকোনো ধরনের value (string, number, boolean, array, object) reactive করে। Template-এ সরাসরি ব্যবহার, JS-এ .value লাগে।

🟢
reactive()

শুধু Object ও Array-এর জন্য। .value লাগে না। Destructure করলে reactivity হারিয়ে যায় — সাবধান!

ref() — সব ধরনের data vue 
<script setup>
import { ref } from 'vue'

// Primitive values
const name    = ref('রাহুল')      // string
const age     = ref(25)           // number
const isAdmin = ref(false)        // boolean

// Array ও Object
const fruits  = ref(['আম', 'কাঁঠাল', 'লিচু'])
const user    = ref({ name: 'করিম', email: 'karim@example.com' })

// JavaScript-এ পরিবর্তন করতে .value ব্যবহার করতে হয়
function updateName() {
  name.value = 'সুমাইয়া'         // ✅ .value লাগবে
}

function addFruit() {
  fruits.value.push('জাম')        // ✅ Array method ব্যবহার
}

function updateUser() {
  user.value.name = 'নাফিসা'      // ✅ Nested property
}
</script>

<template>
  <!-- Template-এ .value লাগে না! Vue স্বয়ংক্রিয়ভাবে unwrap করে -->
  <p>নাম: {{ name }}</p>
  <p>বয়স: {{ age }}</p>
  <p>Admin: {{ isAdmin ? 'হ্যাঁ' : 'না' }}</p>
  <ul>
    <li v-for="fruit in fruits">{{ fruit }}</li>
  </ul>
</template>
reactive() — Object এর জন্য vue 
<script setup>
import { reactive } from 'vue'

// reactive() শুধু Object/Array নেয়
const form = reactive({
  name: '',
  email: '',
  phone: '',
  errors: {}
})

// .value ছাড়াই access করা যায়
function submitForm() {
  if (!form.name) {
    form.errors.name = 'নাম আবশ্যক'
    return
  }
  console.log(form.name, form.email)  // সরাসরি
}

// ⚠️ Destructure করবেন না — reactivity হারাবে!
// const { name } = form  ← ভুল! name আর reactive থাকবে না
</script>

<template>
  <input v-model="form.name" placeholder="আপনার নাম" />
  <span v-if="form.errors.name">{{ form.errors.name }}</span>
</template>
⚠️

পরামর্শ: বেশিরভাগ ক্ষেত্রে ref() ব্যবহার করুন। এটি সব কিছুতে কাজ করে এবং পরে বিভ্রান্তি কম। reactive() শুধু বড় object-এর জন্য ব্যবহার করুন।

Template
Vue Directives — Template-এর যাদু
Beginner

Directives হলো HTML attributes যেগুলো v- দিয়ে শুরু হয়। এগুলো DOM-কে reactive data দিয়ে নিয়ন্ত্রণ করে।

Directiveকী করেউদাহরণ
v-ifConditional rendering (DOM থেকে সরায়)v-if="isLoggedIn"
v-else-ifv-if-এর else ifv-else-if="isGuest"
v-elsev-if false হলে দেখায়v-else
v-showCSS display toggle করেv-show="isVisible"
v-forList render করেv-for="item in items"
v-modelTwo-way data bindingv-model="email"
v-bind / :Attribute binding:href="url"
v-on / @Event listener@click="save"
v-textText content সেট করেv-text="msg"
v-htmlHTML render করে (XSS সাবধান)v-html="htmlContent"
সব Directives একসাথে vue 
<script setup>
import { ref } from 'vue'

const isLoggedIn = ref(true)
const role       = ref('admin')  // 'admin' | 'user' | 'guest'
const showBanner = ref(true)
const students   = ref([
  { id: 1, name: 'আরিফ',    score: 85 },
  { id: 2, name: 'সুমাইয়া', score: 92 },
  { id: 3, name: 'করিম',    score: 78 },
])
const imageUrl = ref('https://picsum.photos/200')
const linkUrl  = ref('https://vuejs.org')
</script>

<template>
  <!-- ✅ v-if / v-else-if / v-else -->
  <div v-if="!isLoggedIn">অনুগ্রহ করে লগইন করুন</div>
  <div v-else-if="role === 'admin'">স্বাগতম, অ্যাডমিন! 👑</div>
  <div v-else>স্বাগতম, ব্যবহারকারী!</div>

  <!-- ✅ v-show (CSS display toggle) -->
  <div v-show="showBanner" class="banner">
    এটা সবসময় DOM-এ থাকে, শুধু লুকানো হয়
  </div>
  <button @click="showBanner = !showBanner">Toggle Banner</button>

  <!-- ✅ v-for — key দেওয়া অবশ্যক! -->
  <ul>
    <li v-for="student in students" :key="student.id">
      {{ student.name }} — স্কোর: {{ student.score }}
    </li>
  </ul>

  <!-- ✅ v-for index সহ -->
  <ol>
    <li v-for="(student, index) in students" :key="student.id">
      {{ index + 1 }}. {{ student.name }}
    </li>
  </ul>

  <!-- ✅ v-bind shorthand → : -->
  <img :src="imageUrl" :alt="'ছবি'" />
  <a :href="linkUrl" target="_blank">Vue.js ওয়েবসাইট</a>

  <!-- ✅ Dynamic class & style -->
  <p :class="{ 'text-red': score < 80, 'text-green': score >= 80 }">
    ফলাফল
  </p>
</template>
🔑

v-if vs v-show: v-if element-টি DOM থেকে সম্পূর্ণ সরিয়ে দেয়। v-show শুধু display: none করে। যদি ঘন ঘন toggle হয়, v-show বেশি দক্ষ। প্রথমবার render costly হলে v-if ব্যবহার করুন।

05
Reactivity
Computed & Watch
Intermediate
🧮
computed()

অন্য reactive data থেকে গণনা করা মান। Cache করা হয় — dependency পরিবর্তন না হলে পুনরায় গণনা হয় না।

👁️
watch()

Reactive data পরিবর্তনে side effect চালায়। API call, localStorage save ইত্যাদির জন্য ব্যবহার করুন।

computed() — গণনা করা মান vue 
<script setup>
import { ref, computed } from 'vue'

const products = ref([
  { name: 'ল্যাপটপ', price: 50000, inStock: true },
  { name: 'মোবাইল', price: 25000, inStock: false },
  { name: 'হেডফোন', price: 3000,  inStock: true },
])

const searchQuery = ref('')
const showOnlyInStock = ref(false)

// ✅ computed — products ও filter পরিবর্তনে auto-update
const filteredProducts = computed(() => {
  return products.value
    .filter(p => !showOnlyInStock.value || p.inStock)
    .filter(p => p.name.includes(searchQuery.value))
})

// ✅ computed দিয়ে মোট দাম
const totalPrice = computed(() =>
  filteredProducts.value.reduce((sum, p) => sum + p.price, 0)
)

// ✅ Writable computed (get + set)
const fullName = computed({
  get: () => `${firstName.value} ${lastName.value}`,
  set: (val) => {
    [firstName.value, lastName.value] = val.split(' ')
  }
})
</script>

<template>
  <input v-model="searchQuery" placeholder="খুঁজুন..." />
  <label>
    <input type="checkbox" v-model="showOnlyInStock" />
    শুধু স্টকে আছে
  </label>

  <p>মোট: {{ filteredProducts.length }}টি পণ্য — টাকা: {{ totalPrice.toLocaleString() }}</p>

  <div v-for="p in filteredProducts" :key="p.name">
    {{ p.name }} — ৳{{ p.price.toLocaleString() }}
    <span v-if="!p.inStock"> (স্টক নেই)</span>
  </div>
</template>
watch() ও watchEffect() vue 
<script setup>
import { ref, watch, watchEffect } from 'vue'

const userId  = ref(1)
const userData = ref(null)
const search  = ref('')

// ✅ watch — নির্দিষ্ট data watch করে
watch(userId, async (newId, oldId) => {
  console.log(`ID পরিবর্তন: ${oldId} → ${newId}`)
  userData.value = await fetchUser(newId)
})

// ✅ Multiple values watch
watch([userId, search], ([newId, newSearch]) => {
  console.log('পরিবর্তন হয়েছে:', newId, newSearch)
})

// ✅ Deep watch (nested object)
const settings = ref({ theme: 'dark', lang: 'bn' })
watch(settings, (newSettings) => {
  localStorage.setItem('settings', JSON.stringify(newSettings))
}, { deep: true })

// ✅ Immediate — প্রথমবার সাথে সাথে চালায়
watch(userId, async (id) => {
  userData.value = await fetchUser(id)
}, { immediate: true })

// ✅ watchEffect — dependencies স্বয়ংক্রিয় detect
watchEffect(async () => {
  // userId.value access করে, তাই userId watch করে
  const data = await fetch(`/api/users/${userId.value}`)
  userData.value = await data.json()
})
</script>

Components — Vue-এর হৃদয়
Intermediate

Component হলো Vue-এর মূল building block। প্রতিটি .vue ফাইল একটি component। এগুলো একে অপরের ভেতরে ব্যবহার করা যায় এবং Props দিয়ে data pass করা যায়।

AlertCard.vue — Child Component vue 
<!-- src/components/AlertCard.vue -->
<script setup>
// Props সংজ্ঞায়িত করা
const props = defineProps({
  title: {
    type: String,
    required: true,
  },
  message: {
    type: String,
    default: 'কোনো বার্তা নেই',
  },
  type: {
    type: String,
    default: 'info',
    validator: (val) => ['success', 'warning', 'error', 'info'].includes(val)
  },
  count: {
    type: Number,
    default: 0
  }
})

// Events সংজ্ঞায়িত করা
const emit = defineEmits(['close', 'action'])

function closeAlert() {
  emit('close')  // Parent-কে জানানো
}

function handleAction() {
  emit('action', props.type)  // Data সহ emit
}
</script>

<template>
  <div :class=`alert alert-${type}`>
    <strong>{{ title }}</strong>
    <p>{{ message }}</p>
    <span v-if="count > 0">{{ count }}টি আইটেম</span>
    <button @click="handleAction">ঠিক আছে</button>
    <button @click="closeAlert">✕</button>
  </div>
</template>

<script setup>
import { ref } from 'vue'

  onBeforeUnmount, onErrorCaptured

const timer   = ref(null)

src/composables/useFetch.js — API Call Composable js 
import { ref, watchEffect } from 'vue'

/**
 * ✅ useFetch — যেকোনো API থেকে data আনে
 * Loading, error সব handle করে
 */
export function useFetch(url) {
  const data    = ref(null)
  const error   = ref(null)
  const loading = ref(false)

  async function fetchData() {
    loading.value = true
    error.value   = null

    try {
      const response = await fetch(url)
      if (!response.ok) throw new Error(`HTTP ${response.status}`)
      data.value = await response.json()
    } catch (err) {
      error.value = err.message
    } finally {
      loading.value = false
    }
  }

  fetchData()  // Immediately fetch

  return { data, error, loading, refetch: fetchData }
}

// ✅ useLocalStorage — Local Storage helper
export function useLocalStorage(key, defaultValue) {
  const stored = localStorage.getItem(key)
  const value = ref(stored ? JSON.parse(stored) : defaultValue)

  watchEffect(() => {
    localStorage.setItem(key, JSON.stringify(value.value))
  })

  return value
}

// ✅ useDebounce — Input delay
export function useDebounce(value, delay = 300) {
  const debouncedValue = ref(value.value)
  let timer

  watchEffect(() => {
    clearTimeout(timer)
    timer = setTimeout(() => {
      debouncedValue.value = value.value
    }, delay)
  })

  return debouncedValue
}

src/stores/authStore.js — Pinia Store js 
// npm install pinia
import { defineStore } from 'pinia'
import { ref, computed } from 'vue'

// ✅ Composition API-style store (সবচেয়ে আধুনিক)
export const useAuthStore = defineStore('auth', () => {
  // === STATE ===
  const user  = ref(null)
  const token = ref(localStorage.getItem('token') || '')

  // === GETTERS (computed) ===
  const isLoggedIn  = computed(() => !!token.value)
  const displayName = computed(() => user.value?.name || 'অতিথি')
  const isAdmin     = computed(() => user.value?.role === 'admin')

  // === ACTIONS ===
  async function login(email, password) {
    try {
      const res = await fetch('/api/auth/login', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ email, password })
      })

      if (!res.ok) throw new Error('লগইন ব্যর্থ')

      const { user: userData, token: newToken } = await res.json()

      user.value  = userData
      token.value = newToken
      localStorage.setItem('token', newToken)

      return { success: true }
    } catch (error) {
      return { success: false, error: error.message }
    }
  }

  function logout() {
    user.value  = null
    token.value = ''
    localStorage.removeItem('token')
  }

  async function fetchProfile() {
    if (!token.value) return
    const res = await fetch('/api/profile', {
      headers: { Authorization: `Bearer ${token.value}` }
    })
    user.value = await res.json()
  }

  return { user, token, isLoggedIn, displayName, isAdmin, login, logout, fetchProfile }
})

<script setup>
import { ref, computed, watch } from 'vue'

// ===== STATE =====
const todos = ref(
  JSON.parse(localStorage.getItem('todos') || '[]')
)
const newTodo      = ref('')
const filter       = ref('all')     // 'all' | 'active' | 'done'
const editingId    = ref(null)
const editingText  = ref('')
let nextId = todos.value.reduce((max, t) => Math.max(max, t.id), 0) + 1

// ===== COMPUTED =====
const filteredTodos = computed(() => {
  switch (filter.value) {
    case 'active': return todos.value.filter(t => !t.done)
    case 'done':   return todos.value.filter(t => t.done)
    default:       return todos.value
  }
})

const activeCount = computed(() => todos.value.filter(t => !t.done).length)
const doneCount   = computed(() => todos.value.filter(t =>  t.done).length)
const allDone     = computed({
  get: () => todos.value.every(t => t.done),
  set: (val) => todos.value.forEach(t => t.done = val)
})

// ===== WATCH — localStorage এ save =====
watch(todos, (val) => {
  localStorage.setItem('todos', JSON.stringify(val))
}, { deep: true })

// ===== ACTIONS =====
function addTodo() {
  if (!newTodo.value.trim()) return
  todos.value.push({
    id:   nextId++,
    text: newTodo.value.trim(),
    done: false,
    createdAt: new Date().toISOString()
  })
  newTodo.value = ''
}

function removeTodo(id) {
  todos.value = todos.value.filter(t => t.id !== id)
}

function startEdit(todo) {
  editingId.value   = todo.id
  editingText.value = todo.text
}

function saveEdit(id) {
  const todo = todos.value.find(t => t.id === id)
  if (todo && editingText.value.trim()) {
    todo.text = editingText.value.trim()
  }
  editingId.value = null
}

function clearDone() {
  todos.value = todos.value.filter(t => !t.done)
}
</script>

<template>
  <div class="todo-app">
    <h1>📝 কাজের তালিকা</h1>

    <!-- Input Area -->
    <div class="input-area">
      <input
        v-model.trim="newTodo"
        @keyup.enter="addTodo"
        placeholder="নতুন কাজ লিখুন... Enter চাপুন"
        autofocus
      />
      <button @click="addTodo" :disabled="!newTodo.trim()">যোগ করুন</button>
    </div>

    <!-- Stats -->
    <div class="stats">
      <span>মোট: {{ todos.length }}</span>
      <span>বাকি: {{ activeCount }}</span>
      <span>শেষ: {{ doneCount }}</span>

      <!-- Toggle All -->
      <label>
        <input type="checkbox" v-model="allDone" />
        সব {{ allDone ? 'বাতিল' : 'সম্পন্ন' }}
      </label>
    </div>

    <!-- Filters -->
    <div class="filters">
      <button
        v-for="f in ['all', 'active', 'done']"
        :key="f"
        :class="{ active: filter === f }"
        @click="filter = f"
      >
        {{ f === 'all' ? 'সব' : f === 'active' ? 'বাকি' : 'শেষ' }}
      </button>
    </div>

    <!-- Todo List -->
    <TransitionGroup name="list" tag="ul">
      <li
        v-for="todo in filteredTodos"
        :key="todo.id"
        :class="{ done: todo.done }"
      >
        <!-- Checkbox -->
        <input type="checkbox" v-model="todo.done" />

        <!-- Inline Edit -->
        <input
          v-if="editingId === todo.id"
          v-model="editingText"
          @keyup.enter="saveEdit(todo.id)"
          @keyup.esc="editingId = null"
          @blur="saveEdit(todo.id)"
          class="edit-input"
          autofocus
        />
        <span v-else @dblclick="startEdit(todo)">
          {{ todo.text }}
        </span>

        <button @click="removeTodo(todo.id)" class="del-btn">🗑️</button>
      </li>
    </TransitionGroup>

    <!-- Empty state -->
    <div v-if="filteredTodos.length === 0" class="empty">
      {{ filter === 'done' ? '✅ কোনো সম্পন্ন কাজ নেই' : '🎉 কোনো কাজ নেই!' }}
    </div>

    <!-- Clear Done -->
    <button
      v-if="doneCount > 0"
      @click="clearDone"
      class="clear-btn"
    >
      সম্পন্ন ({{ doneCount }}টি) মুছুন
    </button>
  </div>
</template>
