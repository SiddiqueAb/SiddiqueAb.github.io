💡 মূল কনসেপ্ট: API কীভাবে কাজ করে?

  • তুমি AI কোম্পানির কাছে একটা API Key নিবে (সাধারণত বিনামূল্যে শুরু করা যায়)
  • তোমার কোড থেকে HTTP request পাঠাবে API endpoint-এ (REST API)
  • AI model তোমার prompt প্রসেস করে response দেবে JSON ফরম্যাটে
  • তুমি সেই response তোমার app-এ display করবে ব্যবহারকারীকে
  • ব্যবহারকারী প্রতি মাসে তোমাকে subscription fee দেবে

OpenAI Service Class তৈরি করো

PHP — app/Services/OpenAIService.php
<?php namespace App\Services; use OpenAI; use Illuminate\Support\Facades\Cache; class OpenAIService { protected $client; public function __construct() { $this->client = OpenAI::client( config('openai.api_key') ); } /** * যেকোনো text generate করো */ public function generateText(string $prompt): string { $response = $this->client->chat()->create([ 'model' => 'gpt-4o', 'messages' => [ [ 'role' => 'system', 'content' => 'তুমি একজন সহায়ক AI assistant।' ], [ 'role' => 'user', 'content' => $prompt ], ], 'max_tokens' => 1000, 'temperature' => 0.7, ]); return $response->choices[0]->message->content; } /** * Streaming response (real-time output) */ public function streamText(string $prompt): iterable { return $this->client->chat()->createStreamed([ 'model' => 'gpt-4o', 'messages' => [['role' => 'user', 'content' => $prompt]], ]); } /** * Image analyze করো */ public function analyzeImage(string $imageUrl, string $question): string { $response = $this->client->chat()->create([ 'model' => 'gpt-4o', 'messages' => [[ 'role' => 'user', 'content' => [ ['type' => 'text', 'text' => $question], ['type' => 'image_url', 'image_url' => ['url' => $imageUrl]], ], ]], ]); return $response->choices[0]->message->content; } }
৪

Controller তৈরি করো

PHP — app/Http/Controllers/AIController.php
<?php namespace App\Http\Controllers; use App\Services\OpenAIService; use Illuminate\Http\Request; use Illuminate\Http\Response; class AIController extends Controller { public function __construct( private OpenAIService $aiService ) {} // Content generate করো public function generate(Request $request): Response { $request->validate([ 'prompt' => 'required|string|max:2000', 'type' => 'required|in:blog,email,social,product', ]); // Token usage track করো per user $user = auth()->user(); if ($user->monthly_tokens_used >= $user->plan->token_limit) { return response()->json([ 'error' => 'মাসিক limit শেষ। Upgrade করুন।' ], 429); } $systemPrompt = match($request->type) { 'blog' => 'তুমি একজন বাংলা ব্লগ লেখক।', 'email' => 'তুমি professional email লেখক।', 'social' => 'তুমি social media content creator।', 'product' => 'তুমি product description লেখক।', }; $content = $this->aiService->generateText( $systemPrompt . "\n\n" . $request->prompt ); // Token count save করো $user->increment('monthly_tokens_used', strlen($content) / 4); return response()->json([ 'content' => $content, 'tokens_left' => $user->plan->token_limit - $user->monthly_tokens_used, ]); } }

Claude API Integration (সরাসরি HTTP)

PHP — Claude API সরাসরি
use Illuminate\Support\Facades\Http; class ClaudeService { private string $apiKey; private string $baseUrl = 'https://api.anthropic.com/v1'; public function __construct() { $this->apiKey = config('services.anthropic.key'); } public function chat(string $message, string $model = 'claude-sonnet-4-5'): string { $response = Http::withHeaders([ 'x-api-key' => $this->apiKey, 'anthropic-version' => '2023-06-01', 'content-type' => 'application/json', ])->post("{$this->baseUrl}/messages", [ 'model' => $model, 'max_tokens' => 2048, 'messages' => [ ['role' => 'user', 'content' => $message] ], ]); if ($response->failed()) { throw new \Exception('Claude API Error: ' . $response->body()); } return $response->json('content.0.text'); } }
ℹ️
Pro Tip: Production-এ সবসময় API calls cache করো। একই prompt দুইবার call করলে অর্থ নষ্ট হয়। Laravel Cache ব্যবহার করো: Cache::remember('ai_'.md5($prompt), 3600, fn() => $ai->generate($prompt))

Node.js, Python, অন্য ভাষায় Integration

📦 Node.js / Express.js

JavaScript
import OpenAI from 'openai'; const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY }); // Express route app.post('/api/generate', async (req, res) => { try { const { prompt } = req.body; const stream = await openai.chat.completions.create({ model: 'gpt-4o', messages: [{ role: 'user', content: prompt }], stream: true, }); // Streaming response res.setHeader('Content-Type', 'text/event-stream'); for await (const chunk of stream) { const text = chunk.choices[0]?.delta?.content || ''; res.write(`data: ${text}\n\n`); } res.end(); } catch (err) { res.status(500).json({ error: err.message }); } });

Python
from openai import AsyncOpenAI from fastapi import FastAPI from fastapi.responses import StreamingResponse import anthropic app = FastAPI() client = AsyncOpenAI() claude = anthropic.Anthropic() @app.post("/generate") async def generate(prompt: str): response = await client.chat.completions.create( model="gpt-4o", messages=[{"role": "user", "content": prompt}], max_tokens=1000 ) return {"content": response.choices[0].message.content} # Claude API @app.post("/claude") async def use_claude(prompt: str): msg = claude.messages.create( model="claude-sonnet-4-5", max_tokens=1024, messages=[{"role": "user", "content": prompt}] ) return {"content": msg.content[0].text}

🔑 যেকোনো Framework-এ AI Integration এর মূল ধাপ

  • API Key .env তে রাখো — কখনো code-এ hardcode করবে না
  • Rate limiting যোগ করো — per user per minute request সীমা দাও
  • Error handling করো — API down থাকলে fallback দাও
  • Response cache করো Redis বা Memcached দিয়ে
  • Token usage database-এ save করো billing-এর জন্য
  • Retry logic যোগ করো (exponential backoff)
