Shopify পরিচিতি

🛍️ Shopify কী?

Shopify হলো বিশ্বের অন্যতম জনপ্রিয় ই-কমার্স প্ল্যাটফর্ম যা ব্যবহার করে যেকোনো ব্যক্তি বা প্রতিষ্ঠান অনলাইনে পণ্য বিক্রি করতে পারে। ২০০৬ সালে টোবিয়াস লুটকে কানাডায় এটি প্রতিষ্ঠা করেন। বর্তমানে বিশ্বজুড়ে ১০ লক্ষেরও বেশি ব্যবসা Shopify ব্যবহার করছে।

একজন ওয়েব ডেভেলপার হিসেবে Shopify শেখা আপনার ক্যারিয়ারে অনেক সুযোগ খুলে দেবে — থিম তৈরি থেকে শুরু করে কাস্টম অ্যাপ ডেভেলপমেন্ট পর্যন্ত।

🏪

ই-কমার্স স্টোর

সহজেই অনলাইন স্টোর তৈরি করুন, পণ্য যোগ করুন এবং পেমেন্ট নিন।

🎨

থিম কাস্টমাইজেশন

Liquid ভাষায় কাস্টম থিম তৈরি করুন এবং স্টোরের ডিজাইন পরিবর্তন করুন।

App Development

📊 Shopify ইকোসিস্টেম

💡
ডেভেলপার হিসেবে Shopify-এ আয়ের সুযোগ একজন দক্ষ Shopify ডেভেলপার প্রতি মাসে $২,০০০ থেকে $১০,০০০+ ডলার আয় করতে পারেন। থিম ডেভেলপমেন্ট, কাস্টম অ্যাপ তৈরি, এবং ফ্রিল্যান্স প্রজেক্টের মাধ্যমে ভালো আয় সম্ভব।

ডেভেলপমেন্ট সেটআপ

💻 প্রয়োজনীয় টুলস

  • Node.js (v18+): Shopify CLI চালাতে প্রয়োজন — nodejs.org থেকে ডাউনলোড করুন
  • Shopify CLI: অফিসিয়াল কমান্ড লাইন টুল — থিম ও অ্যাপ ডেভেলপমেন্টের জন্য
  • VS Code: কোড এডিটর — Shopify Liquid extension ইনস্টল করুন
  • Git: ভার্সন কন্ট্রোলের জন্য প্রয়োজনীয়
  • Shopify Partners Account: ডেভেলপমেন্ট স্টোর তৈরির জন্য ফ্রি অ্যাকাউন্ট
Terminal / Command Line 
# Shopify CLI ইনস্টল করুন
npm install -g @shopify/cli @shopify/theme

# ভার্সন চেক করুন
shopify version

# Shopify CLI দিয়ে লগইন করুন
shopify auth login

# নতুন থিম তৈরি করুন
shopify theme init my-custom-theme

# ডেভেলপমেন্ট সার্ভার চালু করুন
cd my-custom-theme
shopify theme dev --store=your-store.myshopify.com

# থিম পুশ করুন (লাইভ স্টোরে)
shopify theme push

🏷️ প্রোডাক্ট ও কালেকশন

Shopify-এ পণ্য ম্যানেজমেন্টের মূল ধারণাগুলো:

  • Products: প্রতিটি পণ্যের title, description, price, images, variants থাকে
  • Variants: একই পণ্যের বিভিন্ন সাইজ/রঙ (যেমন: Small/Red, Large/Blue)
  • Collections: পণ্যের গ্রুপ — Manual বা Automated হতে পারে
  • Metafields: কাস্টম ডেটা ফিল্ড যোগ করার সুবিধা
  • Tags: পণ্য ফিল্টার ও অর্গানাইজেশনের জন্য

Liquid টেমপ্লেটিং ভাষা

💧 Liquid কী?

Liquid হলো Shopify-এর নিজস্ব টেমপ্লেটিং ভাষা যা Ruby-তে লেখা। এটি দিয়ে HTML-এর মধ্যে dynamic data দেখানো যায়। Liquid-এর তিনটি মূল উপাদান আছে:

  • Objects {{ }}: ডেটা দেখায় (output)
  • Tags {% %}: লজিক নিয়ন্ত্রণ করে (if, for, assign)
  • Filters |: ডেটা রূপান্তর করে (upcase, date, money)

📌 Liquid Objects (অবজেক্ট)

Liquid — Objects 
<!-- প্রোডাক্ট তথ্য -->
<h1>{{ product.title }}</h1>
<p>{{ product.description }}</p>
<span>{{ product.price | money }}</span>

<!-- শপ তথ্য -->
<p>{{ shop.name }}</p>
<a href="{{ shop.url }}">হোমপেজ</a>

<!-- কালেকশন -->
{% for product in collection.products %}
  <div>{{ product.title }}</div>
{% endfor %}

<!-- কার্ট তথ্য -->
<p>মোট পণ্য: {{ cart.item_count }}</p>
<p>মোট মূল্য: {{ cart.total_price | money }}</p>

📌 Liquid Tags (ট্যাগ)

Liquid — Tags (Control Flow) 
<!-- IF কন্ডিশন -->
{% if product.available %}
  <button>কার্টে যোগ করুন</button>
{% elsif product.compare_at_price %}
  <span>সেলে আছে!</span>
{% else %}
  <p>স্টক শেষ</p>
{% endif %}

<!-- FOR লুপ -->
{% for image in product.images limit: 3 %}
  <img src="{{ image | img_url: '400x400' }}" alt="{{ image.alt }}">
{% endfor %}

<!-- ASSIGN ভেরিয়েবল -->
{% assign sale_price = product.price | times: 0.8 %}
<span>ছাড়ের মূল্য: {{ sale_price | money }}</span>

<!-- CASE/WHEN -->
{% case product.type %}
  {% when 'Shirt' %}
    <p>এটি একটি শার্ট</p>
  {% when 'Pants' %}
    <p>এটি একটি প্যান্ট</p>
  {% else %}
    <p>অন্যান্য পোশাক</p>
{% endcase %}

<!-- CAPTURE -->
{% capture product_badge %}
  {% if product.tags contains 'new' %}নতুন{% endif %}
{% endcapture %}
<span class="badge">{{ product_badge }}</span>

📌 Liquid Filters (ফিল্টার)

Liquid — Filters 
<!-- মূল্য ফিল্টার -->
{{ product.price | money }}              <!-- $19.99 -->
{{ product.price | money_with_currency }} <!-- $19.99 USD -->

<!-- ইমেজ ফিল্টার -->
{{ product.featured_image | img_url: '500x500' }}
{{ product.featured_image | img_url: '500x500', crop: 'center' }}
{{ image | image_url: width: 400 }}  <!-- নতুন সিনট্যাক্স -->

<!-- স্ট্রিং ফিল্টার -->
{{ product.title | upcase }}           <!-- TITLE -->
{{ product.title | downcase }}         <!-- title -->
{{ product.description | truncate: 100 }}
{{ product.handle | replace: '-', ' ' }}

<!-- অ্যারে ফিল্টার -->
{{ product.tags | join: ', ' }}        <!-- tag1, tag2, tag3 -->
{{ collection.products | map: 'title' | join: ' | ' }}

<!-- তারিখ ফিল্টার -->
{{ article.published_at | date: '%B %d, %Y' }}
{{ 'now' | date: '%Y-%m-%d' }}

<!-- URL ফিল্টার -->
{{ 'theme.css' | asset_url }}
{{ 'theme.js' | asset_url | script_tag }}
⚠️
গুরুত্বপূর্ণ: Liquid Performance টিপস লুপের ভেতরে include বা ভারী অপারেশন এড়িয়ে চলুন। বেশি নেস্টেড লুপ থিমকে ধীর করতে পারে। সম্ভব হলে {% render %} ব্যবহার করুন (isolated scope সাহায্য করে)।

🎨 theme.liquid — মূল লেআউট

layout/theme.liquid হলো সব পেজের মূল কাঠামো। এখানে <html>, <head>, <body> ট্যাগ থাকে।

layout/theme.liquid 
<!DOCTYPE html>
<html lang="{{ request.locale.iso_code }}">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <!-- SEO Title -->
  <title>
    {{ page_title }}
    {% if current_tags %} &ndash; tagged "{{ current_tags | join: ', ' }}"{% endif %}
    {% unless page_title contains shop.name %} &ndash; {{ shop.name }}{% endunless %}
  </title>

  <!-- Meta Tags -->
  {% if page_description %}
    <meta name="description" content="{{ page_description | escape }}">
  {% endif %}

  <!-- Canonical URL -->
  <link rel="canonical" href="{{ canonical_url }}">

  <!-- CSS ফাইল -->
  {{ 'theme.css' | asset_url | stylesheet_tag }}

  <!-- Shopify প্রয়োজনীয় ট্যাগ -->
  {{ content_for_header }}

</head>

<body>
  {% section 'header' %}

  <main id="MainContent" role="main">
    {{ content_for_layout }}
  </main>

  {% section 'footer' %}

  <!-- JavaScript ফাইল -->
  {{ 'theme.js' | asset_url | script_tag }}

</body>
</html>

Sections ও Blocks

🧩 Sections কী?

Sections হলো Shopify Online Store Editor-এ ড্র্যাগ-ড্রপযোগ্য কম্পোনেন্ট। প্রতিটি Section-এ একটি {% schema %} ব্লক থাকে যা merchant-দের Shopify admin থেকে কাস্টমাইজ করতে দেয়।

sections/hero-banner.liquid 
<!-- Hero Banner Section -->
<div
  class="hero-banner"
  style="background-color: {{ section.settings.bg_color }};"
>
  {% if section.settings.image %}
    <img
      src="{{ section.settings.image | image_url: width: 1200 }}"
      alt="{{ section.settings.image.alt }}"
    >
  {% endif %}

  <div class="hero-content">
    <h2>{{ section.settings.heading }}</h2>
    <p>{{ section.settings.subtext }}</p>

    {% if section.settings.button_label != blank %}
      <a href="{{ section.settings.button_url }}" class="btn">
        {{ section.settings.button_label }}
      </a>
    {% endif %}

// npm install @shopify/shopify-api express
import { shopifyApi, ApiVersion, Session } from '@shopify/shopify-api';
import express from 'express';

const shopify = shopifyApi({
  apiKey: process.env.SHOPIFY_API_KEY,
  apiSecretKey: process.env.SHOPIFY_API_SECRET,
  scopes: ['read_products', 'write_products', 'read_orders'],
  hostName: process.env.HOST,
  apiVersion: ApiVersion.January24,
  isEmbeddedApp: true
});

const app = express();

// OAuth শুরু করার route
app.get('/auth', async (req, res) => {
  await shopify.auth.begin({
    shop: shopify.utils.sanitizeShop(req.query.shop, true),
    callbackPath: '/auth/callback',
    isOnline: false,
    rawRequest: req,
    rawResponse: res
  });
});

// OAuth callback
app.get('/auth/callback', async (req, res) => {
  const callback = await shopify.auth.callback({
    rawRequest: req,
    rawResponse: res
  });
  // Session সেভ করুন ডাটাবেজে
  res.redirect(`/app?shop=${callback.session.shop}`);
});

      }
    }

🌐 Multipass — SSO Integration

Multipass দিয়ে আপনার নিজের সিস্টেমের ইউজারকে সরাসরি Shopify-এ লগইন করানো যায় (Single Sign-On)। শুধুমাত্র Shopify Plus-এ পাওয়া যায়।

Node.js — Multipass Token তৈরি 
import crypto from 'crypto';

function generateMultipassToken(customerData, secret) {
  // দুটি 16-byte key তৈরি করুন
  const hash = crypto.createHash('sha256').update(secret).digest();
  const encryptionKey = hash.slice(0, 16);
  const signingKey = hash.slice(16, 32);

  // Customer data JSON
  const data = JSON.stringify({
    ...customerData,
    created_at: new Date().toISOString()
  });

  // AES-128-CBC এনক্রিপশন
  const iv = crypto.randomBytes(16);
  const cipher = crypto.createCipheriv('aes-128-cbc', encryptionKey, iv);
  const encrypted = Buffer.concat([iv, cipher.update(data), cipher.final()]);

  // HMAC-SHA256 সাইন
  const signature = crypto.createHmac('sha256', signingKey).update(encrypted).digest();

  const token = Buffer.concat([encrypted, signature]).toString('base64url');
  return `https://your-store.myshopify.com/account/login/multipass/${token}`;
}

// ব্যবহার
const url = generateMultipassToken({
  email: 'customer@example.com',
  first_name: 'রহিম',
  last_name: 'মিয়া',
  return_to: '/collections/all'
}, process.env.MULTIPASS_SECRET);
