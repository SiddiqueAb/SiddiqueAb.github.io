01

PSR কী এবং কেন প্রয়োজন?

PSR (PHP Standards Recommendations) হলো PHP-FIG (PHP Framework Interop Group) কর্তৃক প্রণীত কোডিং মানদণ্ড। এগুলো নিয়ম এবং ইন্টারফেস যা বিভিন্ন PHP প্রজেক্ট ও লাইব্রেরিকে একে অপরের সাথে সামঞ্জস্যপূর্ণ করে তোলে।

💡
PHP-FIG প্রতিষ্ঠিত হয়েছিল ২০০৯ সালে। এর সদস্যরা হলেন Laravel, Symfony, Zend, Drupal সহ বড় বড় PHP প্রজেক্টের ডেভেলপাররা। তারা মিলে এই মানদণ্ডগুলো তৈরি করেন যাতে সবাই একটি কমন ভাষায় কথা বলতে পারে।

PSR ব্যবহার না করলে কী সমস্যা?

কোড অসামঞ্জস্যতা

বিভিন্ন লাইব্রেরি ভিন্ন নামকরণ পদ্ধতি অনুসরণ করলে একসাথে ব্যবহার করা কঠিন হয়।

Autoloading জটিলতা

PSR-4 ছাড়া ফাইল লোড করা ম্যানুয়ালি করতে হয়, যা ভুলপ্রবণ এবং সময়সাপেক্ষ।

HTTP লাইব্রেরি বদলানো কঠিন

PSR-7 ছাড়া একটি HTTP লাইব্রেরি থেকে আরেকটিতে যেতে পুরো কোড পরিবর্তন করতে হয়।

দলগত কাজে বিভ্রান্তি

একই দলের বিভিন্ন ডেভেলপার বিভিন্ন স্টাইলে কোড লিখলে রিভিউ ও মেইনটেইন কঠিন হয়।

PSR-1

Basic Coding Standard

মৌলিক কোডিং মানদণ্ড

PHP কোড লেখার সবচেয়ে মৌলিক নিয়মগুলো। প্রতিটি PHP ডেভেলপারকে এগুলো অবশ্যই মেনে চলতে হবে।

✓ Accepted

মূল নিয়মসমূহ

✅
ফাইলে শুধু <?php অথবা <?= ট্যাগ ব্যবহার করতে হবে। <? শর্ট ট্যাগ ব্যবহার নিষিদ্ধ।
বিষয়নিয়মউদাহরণ
PHP ট্যাগশুধু <?php বা <?=<?php echo $name; ?>
এনকোডিংUTF-8 (BOM ছাড়া)—
Class নামStudlyCaps / PascalCaseUserController
Constantসব বড় হাতে, আন্ডারস্কোর দিয়েMAX_SIZE
Method নামcamelCasegetUserName()

PSR-1: সঠিক উদাহরণ — UserController.php 
<?php
// ✅ সঠিক: PSR-1 মেনে চলা

namespace App\Controllers;         // Namespace — StudlyCaps

use App\Models\User;
use App\Services\AuthService;

// ✅ Class নাম: StudlyCaps (PascalCase)
class UserController
{
    // ✅ Constant: সব বড় হাতে + আন্ডারস্কোর
    const MAX_LOGIN_ATTEMPTS = 5;
    const DEFAULT_ROLE = 'user';

    // ✅ Property: camelCase
    private AuthService $authService;

    // ✅ Method নাম: camelCase
    public function getUserById(int $userId): User
    {
        return User::find($userId);
    }

    public function createUser(array $data): User
    {
        $user = new User();
        $user->fill($data);
        $user->save();
        return $user;
    }
}

PSR-3

Logger Interface

লগার ইন্টারফেস

লগিং লাইব্রেরিগুলোর জন্য একটি কমন ইন্টারফেস। Monolog সহ অনেক লাইব্রেরি এটি ইমপ্লিমেন্ট করে।

✓ Accepted

PSR-3 আটটি লগ লেভেল সংজ্ঞায়িত করে, যা RFC 5424 অনুসারে:

🔴 emergency 🟠 alert 🔴 critical 🟡 error 🟡 warning 🔵 notice 🟢 info ⚫ debug
PSR-3: LoggerInterface ও বাস্তব বাস্তবায়ন 
<?php
namespace Psr\Log;

// PSR-3 এর মূল ইন্টারফেস
interface LoggerInterface
{
    // সিস্টেম ব্যবহারের অযোগ্য হলে
    public function emergency(string|Stringable $message, array $context = []): void;

    // তাৎক্ষণিক ব্যবস্থা নিতে হবে
    public function alert(string|Stringable $message, array $context = []): void;

    // গুরুতর সমস্যা
    public function critical(string|Stringable $message, array $context = []): void;

    // Runtime error
    public function error(string|Stringable $message, array $context = []): void;

    // সতর্কতা
    public function warning(string|Stringable $message, array $context = []): void;

    // স্বাভাবিক কিন্তু গুরুত্বপূর্ণ ঘটনা
    public function notice(string|Stringable $message, array $context = []): void;

    // সাধারণ তথ্য
    public function info(string|Stringable $message, array $context = []): void;

    // ডিবাগ তথ্য
    public function debug(string|Stringable $message, array $context = []): void;

    // যেকোনো লেভেলে লগ করা
    public function log($level, string|Stringable $message, array $context = []): void;
}

// ✅ নিজের Logger তৈরি করা
class FileLogger extends AbstractLogger
{
    public function __construct(private string $logFile) {}

    public function log($level, string|Stringable $message, array $context = []): void
    {
        // {placeholder} কে context দিয়ে replace করা
        $message = strtr((string)$message, array_map(
            fn($v) => (string)$v,
            array_combine(
                array_map(fn($k) => '{' . $k . '}', array_keys($context)),
                $context
            )
        ));

        $entry = sprintf(
            "[%s] [%s] %s\n",
            date('Y-m-d H:i:s'),
            strtoupper($level),
            $message
        );

        file_put_contents($this->logFile, $entry, FILE_APPEND);
    }
}

// ব্যবহার
$logger = new FileLogger('/logs/app.log');
$logger->info('ব্যবহারকারী লগইন করেছে: {username}', ['username' => 'রাহুল']);
$logger->error('ডাটাবেজ সংযোগ ব্যর্থ: {error}', ['error' => $e->getMessage()]);
PSR-4

Autoloading Standard

অটোলোডিং মানদণ্ড

PHP ক্লাস ফাইলগুলো স্বয়ংক্রিয়ভাবে লোড করার নিয়ম। এটি সবচেয়ে গুরুত্বপূর্ণ PSR। Composer এটি ব্যবহার করে।

✓ Accepted
🔑
মূল নিয়ম: Fully Qualified Class Name (FQCN) → ফাইল পাথের সাথে মিলতে হবে।
যেমন: App\Http\Controllers\UserController → src/Http/Controllers/UserController.php
PSR-4: composer.json কনফিগারেশন 
{
    "autoload": {
        "psr-4": {
            "App\\": "src/",
            "Tests\\": "tests/"
        }
    }
}

// ফাইল কাঠামো:
// src/Http/Controllers/UserController.php
// → namespace App\Http\Controllers;
//
// src/Models/User.php
// → namespace App\Models;
//
// src/Services/UserService.php
// → namespace App\Services;
PSR-4: নিজের Autoloader তৈরি (Composer ছাড়া) 
<?php
// নিজস্ব PSR-4 Autoloader

class Psr4Autoloader
{
    private array $prefixes = [];

    /**
     * Namespace prefix এবং base directory নিবন্ধন করা
     */
    public function addNamespace(string $prefix, string $baseDir): void
    {
        // prefix-এর শেষে namespace separator নিশ্চিত করা
        $prefix = trim($prefix, '\\') . '\\';

        // base directory-র শেষে slash নিশ্চিত করা
        $baseDir = rtrim($baseDir, DIRECTORY_SEPARATOR) . '/';

        $this->prefixes[$prefix][] = $baseDir;
    }

    /**
     * ক্লাস নাম থেকে ফাইল খোঁজা ও লোড করা
     */
    public function loadClass(string $class): bool
    {
        // namespace prefix খোঁজা
        $prefix = $class;

        while (($pos = strrpos($prefix, '\\')) !== false) {
            $prefix        = substr($class, 0, $pos + 1);
            $relativeClass = substr($class, $pos + 1);

            if (isset($this->prefixes[$prefix])) {
                foreach ($this->prefixes[$prefix] as $baseDir) {
                    // ফাইল পাথ তৈরি: namespace separator → directory separator
                    $file = $baseDir . str_replace('\\', '/', $relativeClass) . '.php';

                    if (file_exists($file)) {
                        require $file;
                        return true;
                    }
                }
            }

            $prefix = rtrim($prefix, '\\');
        }

        return false;
    }

    /**
     * SPL autoloader হিসেবে রেজিস্টার করা
     */
    public function register(): void
    {
        spl_autoload_register([$this, 'loadClass']);
    }
}

// ব্যবহার:
$loader = new Psr4Autoloader();
$loader->addNamespace('App', __DIR__ . '/src');
$loader->addNamespace('Tests', __DIR__ . '/tests');
$loader->register();

// এখন স্বয়ংক্রিয়ভাবে লোড হবে:
$user = new App\Models\User();            // → src/Models/User.php
$ctrl = new App\Http\Controllers\Home(); // → src/Http/Controllers/Home.php
PSR-6

Caching Interface

ক্যাশিং ইন্টারফেস (পূর্ণাঙ্গ)

CacheItemInterface ও CacheItemPoolInterface। PSR-16 এর তুলনায় বেশি বৈশিষ্ট্যসম্পন্ন কিন্তু জটিল।

✓ Accepted
PSR-6: Cache Implementation 
<?php
use Psr\Cache\CacheItemInterface;
use Psr\Cache\CacheItemPoolInterface;

// CacheItem বাস্তবায়ন
class CacheItem implements CacheItemInterface
{
    private bool $isHit = false;
    private $value;
    private ?\DateTimeInterface $expiresAt = null;

    public function __construct(private string $key) {}

    public function getKey(): string       { return $this->key; }
    public function get(): mixed           { return $this->value; }
    public function isHit(): bool          { return $this->isHit; }

    public function set(mixed $value): static
    {
        $this->value = $value;
        $this->isHit = true;
        return $this;
    }

    public function expiresAt(?\DateTimeInterface $expiration): static
    {
        $this->expiresAt = $expiration;
        return $this;
    }

    public function expiresAfter(int|\DateInterval|null $time): static
    {
        if (is_int($time)) {
            $this->expiresAt = new \DateTime('+' . $time . ' seconds');
        }
        return $this;
    }
}

// ব্যবহার:
$pool = new FileCachePool('/tmp/cache');
$item = $pool->getItem('user_profile_42');

if (!$item->isHit()) {
    $userData = $db->fetchUser(42);
    $item->set($userData)->expiresAfter(3600); // ১ ঘণ্টা
    $pool->save($item);
}

$user = $item->get(); // cache থেকে বা DB থেকে
PSR-7

HTTP Message Interface

HTTP বার্তা ইন্টারফেস

HTTP Request এবং Response এর জন্য কমন ইন্টারফেস। Immutable অবজেক্ট প্যাটার্ন ব্যবহার করে।

✓ Accepted
⚠️
গুরুত্বপূর্ণ: PSR-7 অবজেক্টগুলো immutable (অপরিবর্তনীয়)। কোনো পরিবর্তন করলে নতুন অবজেক্ট তৈরি হয়, পুরোনোটি অপরিবর্তিত থাকে।
PSR-7: HTTP Request & Response বাস্তবায়ন 
<?php
use Psr\Http\Message\RequestInterface;
use Psr\Http\Message\ResponseInterface;
use Psr\Http\Message\StreamInterface;

// ✅ সঠিক: Immutable পরিবর্তন
$request = new ServerRequest('GET', 'https://api.example.com/users');

// withHeader → নতুন অবজেক্ট দেয়, পুরোনোটি অপরিবর্তিত
$newRequest = $request->withHeader('Authorization', 'Bearer token123')
                         ->withHeader('Accept', 'application/json');

// Request বাস্তবায়ন
class Request implements RequestInterface
{
    private array $headers = [];
    private string $method;
    private \Psr\Http\Message\UriInterface $uri;

    public function getMethod(): string
    {
        return strtoupper($this->method);
    }

    // Immutable: নতুন instance ফেরত দেয়
    public function withMethod(string $method): static
    {
        $clone = clone $this;
        $clone->method = strtoupper($method);
        return $clone;
    }

    public function withHeader(string $name, $value): static
    {
        $clone = clone $this;
        $clone->headers[strtolower($name)] = [$value];
        return $clone;
    }

    public function getHeader(string $name): array
    {
        return $this->headers[strtolower($name)] ?? [];
    }
}

// Response তৈরি
class Response implements ResponseInterface
{
    public function __construct(
        private int $statusCode = 200,
        private array $headers = [],
        private ?StreamInterface $body = null,
        private string $reasonPhrase = 'OK',
        private string $protocol = '1.1'
    ) {}

    public function getStatusCode(): int      { return $this->statusCode; }
    public function getReasonPhrase(): string  { return $this->reasonPhrase; }

    public function withStatus(int $code, string $phrase = ''): static
    {
        $clone = clone $this;
        $clone->statusCode = $code;
        $clone->reasonPhrase = $phrase ?: $this->getDefaultPhrase($code);
        return $clone;
    }

    private function getDefaultPhrase(int $code): string
    {
        return match($code) {
            200 => 'OK', 201 => 'Created', 400 => 'Bad Request',
            401 => 'Unauthorized', 403 => 'Forbidden',
            404 => 'Not Found', 500 => 'Internal Server Error',
            default => ''
        };
    }
}
PSR-11

Container Interface

Dependency Injection Container

Dependency Injection Container-এর জন্য কমন ইন্টারফেস। Laravel-এর IoC Container, Symfony-র DI Container এটি মেনে চলে।

✓ Accepted
PSR-11: DI Container সম্পূর্ণ বাস্তবায়ন 
<?php
use Psr\Container\ContainerInterface;
use Psr\Container\NotFoundExceptionInterface;
use Psr\Container\ContainerExceptionInterface;

// ✅ সম্পূর্ণ DI Container
class Container implements ContainerInterface
{
    private array $bindings  = [];  // binding definitions
    private array $instances = [];  // singleton instances

    /**
     * সার্ভিস বাঁধা (bind)
     */
    public function bind(string $abstract, \Closure|string|null $concrete = null): void
    {
        $this->bindings[$abstract] = $concrete ?? $abstract;
    }

    /**
     * Singleton হিসেবে বাঁধা
     */
    public function singleton(string $abstract, \Closure|string|null $concrete = null): void
    {
        $this->bind($abstract, function($c) use ($abstract, $concrete) {
            if (!isset($this->instances[$abstract])) {
                $this->instances[$abstract] = ($concrete ?? fn() => new $abstract)($c);
            }
            return $this->instances[$abstract];
        });
    }

    /**
     * সার্ভিস resolve করা (PSR-11)
     */
    public function get(string $id): mixed
    {
        if (isset($this->bindings[$id])) {
            $concrete = $this->bindings[$id];
            return $concrete instanceof \Closure
                ? $concrete($this)
                : $this->resolve($concrete);
        }

        if (class_exists($id)) {
            return $this->resolve($id);
        }

        throw new NotFoundException("সার্ভিস পাওয়া যায়নি: {$id}");
    }

    public function has(string $id): bool
    {
        return isset($this->bindings[$id]) || class_exists($id);
    }

    /**
     * Reflection দিয়ে স্বয়ংক্রিয় Dependency Injection
     */
    private function resolve(string $class): mixed
    {
        $reflector = new \ReflectionClass($class);

        if (!$reflector->isInstantiable()) {
            throw new ContainerException("{$class} instantiable নয়");
        }

        $constructor = $reflector->getConstructor();

        if (!$constructor) {
            return new $class();
        }

        // Constructor-এর প্রতিটি parameter স্বয়ংক্রিয়ভাবে resolve করা
        $dependencies = array_map(
            fn($param) => $this->resolveDependency($param),
            $constructor->getParameters()
        );

        return $reflector->newInstanceArgs($dependencies);
    }

    private function resolveDependency(\ReflectionParameter $param): mixed
    {
        $type = $param->getType();

        if ($type && !$type->isBuiltin()) {
            return $this->get($type->getName());
        }

        if ($param->isDefaultValueAvailable()) {
            return $param->getDefaultValue();
        }

        throw new ContainerException("Dependency resolve করা যাচ্ছে না: {$param->getName()}");
    }
}

// ব্যবহার:
$container = new Container();

// Interface → Implementation বাঁধা

// Singleton
$container->singleton('db', fn() => new Database(getenv('DB_DSN')));

// স্বয়ংক্রিয় DI সহ UserController তৈরি

class UserLoginEvent implements StoppableEventInterface

// ✅ ListenerProvider
class ListenerProvider implements ListenerProviderInterface
{
    private array $listeners = [];

    public function on(string $eventClass, callable $listener): void
    {
        $this->listeners[$eventClass][] = $listener;
    }

    public function getListenersForEvent(object $event): iterable
    {
        return $this->listeners[$event::class] ?? [];
    }
}

// ✅ EventDispatcher
class EventDispatcher implements EventDispatcherInterface
{
    public function __construct(private ListenerProviderInterface $provider) {}

    public function dispatch(object $event): object
    {
        foreach ($this->provider->getListenersForEvent($event) as $listener) {
            if ($event instanceof StoppableEventInterface && $event->isPropagationStopped()) {
                break;
            }
            $listener($event);
        }
        return $event;
    }
}

✓ Accepted
PSR-15: Middleware Pipeline 
<?php
use Psr\Http\Message\{ResponseInterface, ServerRequestInterface};
use Psr\Http\Server\{MiddlewareInterface, RequestHandlerInterface};

// ✅ Authentication Middleware
class AuthMiddleware implements MiddlewareInterface
{
    public function process(
        ServerRequestInterface $request,
        RequestHandlerInterface $handler
    ): ResponseInterface {
        $token = $request->getHeaderLine('Authorization');

        if (empty($token) || !$this->isValidToken($token)) {
            return new JsonResponse(['error' => 'Unauthorized'], 401);
        }

        // token valid → পরের handler-এ যাও
        return $handler->handle($request);
    }

    private function isValidToken(string $token): bool
    {
        return str_starts_with($token, 'Bearer ') && strlen($token) > 10;
    }
}

// ✅ Logging Middleware
class LoggingMiddleware implements MiddlewareInterface
{
    public function process(
        ServerRequestInterface $request,
        RequestHandlerInterface $handler
    ): ResponseInterface {
        $start = microtime(true);

        $response = $handler->handle($request); // পরবর্তী middleware চালাও

        $time = round((microtime(true) - $start) * 1000, 2);

        error_log(sprintf(
            "%s %s → %d [%sms]",
            $request->getMethod(),
            $request->getUri()->getPath(),
            $response->getStatusCode(),
            $time
        ));

        return $response;
    }
}

// ✅ Middleware Pipeline (Queue)
class MiddlewarePipeline implements RequestHandlerInterface
{
    private array $middleware = [];

    public function pipe(MiddlewareInterface $middleware): static
    {
        $clone = clone $this;
        $clone->middleware[] = $middleware;
        return $clone;
    }

    public function handle(ServerRequestInterface $request): ResponseInterface
    {
        $handler = array_reduce(
            array_reverse($this->middleware),
            fn($carry, $mw) => new class($mw, $carry) implements RequestHandlerInterface {
                public function __construct(private $mw, private $next) {}
                public function handle(ServerRequestInterface $req): ResponseInterface {
                    return $this->mw->process($req, $this->next);
                }
            },
            $this->finalHandler
        );
        return $handler->handle($request);
    }
}

// Pipeline তৈরি ও ব্যবহার:
$pipeline = (new MiddlewarePipeline($finalHandler))
    ->pipe(new LoggingMiddleware())
    ->pipe(new AuthMiddleware())
    ->pipe(new CorsMiddleware());

$response = $pipeline->handle($serverRequest);

Simple Cache Interface

PSR-6 এর সরল সংস্করণ। get/set/delete/clear/has — মাত্র ৫টি মেথড।

✓ Accepted
PSR-16: Simple Cache
<?php
use Psr\SimpleCache\CacheInterface;

class ArrayCache implements CacheInterface
{
    private array $store = [];
    private array $expiry = [];

    public function get(string $key, mixed $default = null): mixed
    {
        if (!$this->has($key)) return $default;
        return $this->store[$key];
    }

    public function set(string $key, mixed $value, int|null $ttl = null): bool
    {
        $this->store[$key] = $value;
        if ($ttl) $this->expiry[$key] = time() + $ttl;
        return true;
    }

    public function has(string $key): bool
    {
        if (!isset($this->store[$key])) return false;
        if (isset($this->expiry[$key]) && $this->expiry[$key] < time()) {
            unset($this->store[$key], $this->expiry[$key]);
            return false;
        }
        return true;
    }

    public function delete(string $key): bool
    {
        unset($this->store[$key], $this->expiry[$key]);
        return true;
    }

    public function clear(): bool
    {
        $this->store = $this->expiry = [];
        return true;
    }

    public function getMultiple(iterable $keys, mixed $default = null): iterable
    {
        foreach ($keys as $key) yield $key => $this->get($key, $default);
    }

    public function setMultiple(iterable $values, int|null $ttl = null): bool
    {
        foreach ($values as $k => $v) $this->set($k, $v, $ttl);
        return true;
    }

    public function deleteMultiple(iterable $keys): bool
    {
        foreach ($keys as $key) $this->delete($key);
        return true;
    }
}

declare(strict_types=1);

namespace Framework;

use Framework\Container\Container;
use Framework\Http\{Request, Response};

use Framework\Middleware\Pipeline;
use Framework\Log\FileLogger;
use Framework\Cache\FileCache;
use Psr\Http\Message\{ResponseInterface, ServerRequestInterface};
use Psr\Http\Server\MiddlewareInterface;

class Application
{
    private Container $container;
    private Router $router;
    private array $middlewares = [];

    public function __construct(private string $basePath = '')
    {
        $this->container = new Container();
        $this->router    = new Router();
        $this->registerCoreServices();
    }

    private function registerCoreServices(): void
    {
        // Logger (PSR-3)
        $this->container->singleton(
            \Psr\Log\LoggerInterface::class,
            fn() => new FileLogger($this->basePath . '/storage/logs/app.log')
        );

        // Cache (PSR-16)
        $this->container->singleton(
            \Psr\SimpleCache\CacheInterface::class,

        );

        // Router নিজেকে বাঁধা
        $this->container->singleton(Router::class, fn() => $this->router);
    }

    // ===== Route নিবন্ধন =====
    public function get(string $path, array|callable $handler): static
    {
        $this->router->add('GET', $path, $handler);
        return $this;
    }

    public function post(string $path, array|callable $handler): static
    {
        $this->router->add('POST', $path, $handler);
        return $this;
    }

    public function put(string $path, array|callable $handler): static
    {
        $this->router->add('PUT', $path, $handler);
        return $this;
    }

    public function delete(string $path, array|callable $handler): static
    {
        $this->router->add('DELETE', $path, $handler);
        return $this;
    }

    // Middleware যোগ করা (PSR-15)
    public function use(MiddlewareInterface $middleware): static
    {
        $this->middlewares[] = $middleware;
        return $this;
    }

    // Container থেকে সার্ভিস পাওয়া
    public function make(string $abstract): mixed
    {
        return $this->container->get($abstract);
    }

    // Application চালু করা
    public function run(): void
    {
        try {
            // PSR-7 Request তৈরি
            $request = Request::fromGlobals();

            // Pipeline দিয়ে handle করা
            $pipeline = new Pipeline($this->container, $this->router);

            foreach ($this->middlewares as $middleware) {
                $pipeline->pipe($middleware);
            }

            $response = $pipeline->handle($request);
            $this->send($response);

        } catch (\Throwable $e) {
            $this->handleException($e);
        }
    }

    private function send(ResponseInterface $response): void
    {
        http_response_code($response->getStatusCode());

        foreach ($response->getHeaders() as $name => $values) {
            foreach ($values as $value) {
                header("{$name}: {$value}", false);
            }
        }

        echo (string)$response->getBody();
    }

    private function handleException(\Throwable $e): void
    {
        try {
            $this->make(\Psr\Log\LoggerInterface::class)->critical(
                'Unhandled exception: {message}',
                ['message' => $e->getMessage(), 'trace' => $e->getTraceAsString()]
            );
        } catch (\Throwable) {}

        http_response_code(500);
        header('Content-Type: application/json');
        echo json_encode(['error' => 'Internal Server Error', 'message' => $e->getMessage()]);
    }

২. Router.php — রাউটিং

src/Routing/Router.php
<?php

declare(strict_types=1);

namespace Framework\Routing;

use Psr\Http\Message\ServerRequestInterface;

class Router
{
    private array $routes = [];

    public function add(string $method, string $path, array|callable $handler): void
    {
        $this->routes[] = new Route(strtoupper($method), $path, $handler);
    }

    public function match(ServerRequestInterface $request): ?RouteMatch
    {
        $method = $request->getMethod();
        $path   = $request->getUri()->getPath();

        foreach ($this->routes as $route) {
            if ($route->getMethod() !== $method) continue;

            $params = $route->match($path);

            if ($params !== null) {
                return new RouteMatch($route->getHandler(), $params);
            }
        }

        return null;
    }
}

class Route
{
    private string $pattern;

    public function __construct(
        private string $method,
        private string $path,
        private array|callable $handler,
    ) {
        // /users/{id} → /users/(?P<id>[^/]+)
        $this->pattern = '#^' . preg_replace(
            '/\{(\w+)\}/',
            '(?P<$1>[^/]+)',
            $path
        ) . '$#';
    }

    public function getMethod(): string           { return $this->method; }
    public function getHandler(): array|callable  { return $this->handler; }

    public function match(string $path): ?array
    {
        if (preg_match($this->pattern, $path, $matches)) {
            return array_filter($matches, 'is_string', ARRAY_FILTER_USE_KEY);
        }
        return null;
    }
}

class RouteMatch
{
    public function __construct(
        public readonly array|callable $handler,
        public readonly array $params,
    ) {}
}

৩. Pipeline.php — Middleware

src/Middleware/Pipeline.php
<?php

declare(strict_types=1);

namespace Framework\Middleware;

use Framework\Container\Container;
use Framework\Http\{JsonResponse, Response};
use Framework\Routing\Router;
use Psr\Http\Message\{ResponseInterface, ServerRequestInterface};
use Psr\Http\Server\{MiddlewareInterface, RequestHandlerInterface};

class Pipeline implements RequestHandlerInterface
{
    private array $middlewares = [];

    public function __construct(
        private Container $container,
        private Router $router,
    ) {}

    public function pipe(MiddlewareInterface $middleware): static
    {
        $this->middlewares[] = $middleware;
        return $this;
    }

    public function handle(ServerRequestInterface $request): ResponseInterface
    {
        // Final handler: Router দিয়ে dispatch
        $finalHandler = new class($this->container, $this->router) implements RequestHandlerInterface
        {
            public function __construct(
                private Container $container,
                private Router $router,
            ) {}

            public function handle(ServerRequestInterface $request): ResponseInterface
            {
                $match = $this->router->match($request);

                if (!$match) {
                    return new JsonResponse(
                        ['error' => 'Route পাওয়া যায়নি'],
                        404
                    );
                }

                // Route params request-এ যোগ করা
                foreach ($match->params as $key => $value) {
                    $request = $request->withAttribute($key, $value);
                }

                return $this->callHandler($match->handler, $request);
            }

            private function callHandler(array|callable $handler, ServerRequestInterface $request): ResponseInterface
            {
                if (is_callable($handler)) {
                    return $handler($request);
                }

                // [ControllerClass, method] → Container দিয়ে তৈরি
                [$class, $method] = $handler;
                $controller = $this->container->get($class);
                return $controller->$method($request);
            }
        };

        // Middleware stack তৈরি (শেষ থেকে শুরু)
        $handler = array_reduce(
            array_reverse($this->middlewares),
            static function (RequestHandlerInterface $carry, MiddlewareInterface $middleware): RequestHandlerInterface {
                return new class($middleware, $carry) implements RequestHandlerInterface {
                    public function __construct(
                        private MiddlewareInterface $middleware,
                        private RequestHandlerInterface $next,
                    ) {}

                    public function handle(ServerRequestInterface $request): ResponseInterface
                    {
                        return $this->middleware->process($request, $this->next);
                    }
                };
            },
            $finalHandler
        );

        return $handler->handle($request);
    }
}

class UserController
{

    public function __construct(

    /**
     * GET /users — সব users
     */

    {

        $cacheKey = "users_page_{$page}_limit_{$limit}";

        // PSR-16 Cache চেক করা
        if ($this->cache->has($cacheKey)) {
            $this->logger->debug('Cache hit: {key}', ['key' => $cacheKey]);
            return new JsonResponse($this->cache->get($cacheKey));
        }

        $users = $this->userService->paginate($page, $limit);

        // Cache-এ সংরক্ষণ (৫ মিনিট)
        $this->cache->set($cacheKey, $users, 300);

        $this->logger->info('Users listed: page {page}', ['page' => $page]);
        return new JsonResponse($users);
    }

    /**
     * GET /users/{id} — একটি user
     */
    public function show(ServerRequestInterface $request): ResponseInterface
    {
        $id   = (int) $request->getAttribute('id');
        $user = $this->userService->findById($id);

        if (!$user) {
            return new JsonResponse(['error' => 'User পাওয়া যায়নি'], 404);
        }

        return new JsonResponse($user);
    }

    /**
     * POST /users — নতুন user তৈরি
     */
    public function store(ServerRequestInterface $request): ResponseInterface
    {
        $data = $request->getParsedBody();

        if (empty($data['name']) || empty($data['email'])) {
            return new JsonResponse(['error' => 'name এবং email আবশ্যক'], 422);
        }

        try {
            $user = $this->userService->create($data);
            $this->logger->info('User তৈরি হয়েছে: {email}', ['email' => $data['email']]);
            return new JsonResponse($user, 201);
        } catch (\Exception $e) {
            $this->logger->error('User তৈরিতে সমস্যা: {error}', ['error' => $e->getMessage()]);
            return new JsonResponse(['error' => $e->getMessage()], 400);
        }
    }

    /**
     * DELETE /users/{id} — user মুছে ফেলা
     */
    public function destroy(ServerRequestInterface $request): ResponseInterface
    {
        $id = (int) $request->getAttribute('id');
        $this->userService->delete($id);
        $this->logger->warning('User মুছে ফেলা হয়েছে: {id}', ['id' => $id]);
        return new JsonResponse(['success' => true]);
    }
}
