// ০১ — FOUNDATION //

ব্লকচেইন কী? কীভাবে কাজ করে?

ব্লকচেইন হলো একটি বিকেন্দ্রীভূত ডিজিটাল খাতাবই (Distributed Ledger) যেখানে তথ্য একবার লেখা হলে পরিবর্তন করা প্রায় অসম্ভব।

🔗 ব্লকচেইনের সংজ্ঞা ও ইতিহাস

২০০৮ সালে সাতোশি নাকামোতো (ছদ্মনাম) নামক এক রহস্যময় ব্যক্তি বা গোষ্ঠী "Bitcoin: A Peer-to-Peer Electronic Cash System" শিরোনামে একটি হোয়াইটপেপার প্রকাশ করেন। এটি ছিল ব্লকচেইনের জন্ম মুহূর্ত।

ব্লকচেইন কল্পনা করুন একটি Google Spreadsheet এর মতো — কিন্তু এই শীটটি লক্ষ লক্ষ কম্পিউটারে একসাথে সংরক্ষিত আছে এবং কেউ একা মালিক নয়। প্রতিটি "ব্লক" একটি পাতার মতো এবং সব পাতা মিলে তৈরি হয় "চেইন"।

ব্লকচেইনে একবার ডেটা লেখা হলে সেটি পরিবর্তন করতে হলে পুরো নেটওয়ার্কের ৫১% এর বেশি কম্পিউটারের সম্মতি লাগবে — যা বাস্তবে প্রায় অসম্ভব।
📦

ব্লক (Block)

প্রতিটি ব্লকে থাকে: লেনদেনের তথ্য (Transaction Data), আগের ব্লকের হ্যাশ (Previous Hash), বর্তমান ব্লকের হ্যাশ (Current Hash) এবং টাইমস্ট্যাম্প। এই চারটি উপাদান মিলে একটি ব্লক গঠিত হয়।

⛓️

চেইন (Chain)

প্রতিটি ব্লক আগের ব্লকের হ্যাশ ধারণ করে, তাই সব ব্লক একটি শৃঙ্খলে আবদ্ধ। একটি ব্লক পরিবর্তন করলে তার পরের সব ব্লকের হ্যাশ ভেঙে পড়ে — এটাই ব্লকচেইনের নিরাপত্তার মূল রহস্য।

🌐

নোড (Node)

নেটওয়ার্কে সংযুক্ত প্রতিটি কম্পিউটারকে নোড বলে। প্রতিটি নোড ব্লকচেইনের সম্পূর্ণ কপি রাখে। কোনো লেনদেন হলে সব নোড যাচাই করে এবং সংখ্যাগরিষ্ঠ সম্মত হলে তা যোগ হয়।

#️⃣

হ্যাশিং (Hashing)

SHA-256 অ্যালগরিদম ব্যবহার করে যেকোনো ডেটাকে একটি নির্দিষ্ট দৈর্ঘ্যের অক্ষরক্রমে রূপান্তর করা হয়। ডেটায় এক বিট পরিবর্তন হলেও হ্যাশ সম্পূর্ণ আলাদা হয়ে যায়।

🤝

কনসেনসাস মেকানিজম

নেটওয়ার্কে একমত হওয়ার পদ্ধতি। দুটি প্রধান ধরন: Proof of Work (PoW) — গণনার মাধ্যমে প্রমাণ, এবং Proof of Stake (PoS) — সম্পদ জমার মাধ্যমে প্রমাণ।

🔐

ক্রিপ্টোগ্রাফি

Public Key ও Private Key জুটি ব্যবহার করে লেনদেন সুরক্ষিত করা হয়। Public Key হলো আপনার ঠিকানা (সবাই দেখতে পায়), Private Key হলো আপনার পাসওয়ার্ড (গোপন রাখতে হবে)।

💡 ব্লকচেইনের প্রকারভেদ

  • Public Blockchain: সবার জন্য উন্মুক্ত — Bitcoin, Ethereum. যে কেউ যোগ দিতে পারে।
  • Private Blockchain: শুধুমাত্র অনুমোদিত ব্যক্তি প্রবেশ করতে পারে — ব্যাংক বা কর্পোরেট ব্যবহারে।
  • Consortium Blockchain: একাধিক সংস্থা মিলে পরিচালনা করে — Hyperledger, R3 Corda.
  • Hybrid Blockchain: Public ও Private-এর মিশ্রণ — Dragonchain ব্যবহার করে।
// একটি সাধারণ ব্লকের গঠন (JavaScript)
class Block {
  constructor(index, timestamp, data, previousHash = '') {
    this.index = index; // ব্লক নম্বর
    this.timestamp = timestamp; // কখন তৈরি হয়েছে
    this.data = data; // লেনদেনের তথ্য
    this.previousHash = previousHash; // আগের ব্লকের হ্যাশ
    this.hash = this.calculateHash(); // বর্তমান হ্যাশ
  }

  calculateHash() {
    // SHA-256 দিয়ে হ্যাশ তৈরি
    return SHA256(index + previousHash + timestamp + data);
  }
}

ক্রিপ্টো মাইনিং — ডিজিটাল সোনা খোঁজা

⛏️ মাইনিং কীভাবে কাজ করে?

মাইনিং হলো একটি প্রতিযোগিতামূলক গাণিতিক সমস্যা সমাধানের প্রক্রিয়া। যে কম্পিউটার আগে সমাধান করতে পারে সে ব্লক যোগ করার অধিকার পায় এবং পুরস্কার হিসেবে ক্রিপ্টো পায়।

বিটকয়েন মাইনিং-এ প্রতি ব্লকের পুরস্কার এখন ৩.১২৫ BTC (২০২৪ Halving পরবর্তী)। প্রতিটি ব্লক গড়ে ১০ মিনিটে মাইন হয়।
💻

CPU Mining

সাধারণ প্রসেসর দিয়ে মাইনিং। বিটকয়েনের শুরুতে ছিল। এখন বিটকয়েনে অলাভজনক, তবে কিছু altcoin-এ এখনো সম্ভব।

🎮

GPU Mining

গেমিং গ্রাফিক্স কার্ড দিয়ে মাইনিং। Ethereum মাইনিং-এ জনপ্রিয় ছিল। ETH এর Merge-এর পর GPU মাইনার অন্য কয়েনে গেছে।

🔧

ASIC Mining

বিশেষভাবে তৈরি মাইনিং যন্ত্র। বিটকয়েনে সবচেয়ে কার্যকর। Antminer S21 প্রতি সেকেন্ডে ২০০ TH/s হ্যাশ করে।

🏊

Mining Pool

একাধিক মাইনার একসাথে কাজ করে পুরস্কার ভাগ করে নেয়। একা মাইন করার চেয়ে নিয়মিত আয়ের সুযোগ। F2Pool, Antpool, ViaBTC জনপ্রিয়।

☁️

Cloud Mining

নিজের হার্ডওয়্যার ছাড়া অনলাইনে মাইনিং শেয়ার কেনা। অনেক ক্ষেত্রে স্ক্যামের ঝুঁকি আছে। NiceHash, Genesis Mining জনপ্রিয়।

🌱

Staking (PoS)

ক্রিপ্টো জমা রেখে নেটওয়ার্ক যাচাইকারী হওয়া। কোনো বিদ্যুৎ খরচ নেই। Ethereum-এ ৩২ ETH স্টেক করতে হয়। APY সাধারণত ৩-১০%।

⚡ মাইনিং ও পরিবেশ বিতর্ক

বিটকয়েন মাইনিং প্রতি বছর প্রায় ১৫০ TWh বিদ্যুৎ ব্যবহার করে — এটি আর্জেন্টিনার মোট বিদ্যুৎ খরচের সমতুল্য। Ethereum PoS-এ যাওয়ার পর ৯৯.৯৫% কম বিদ্যুৎ ব্যবহার করছে।

  • বিটকয়েন মাইনারদের ৫৮%+ নবায়নযোগ্য শক্তি ব্যবহার করে (২০২৩ তথ্য)
  • চীন মাইনিং নিষিদ্ধ করেছে, তবে ভূগর্ভস্থভাবে চলছে
  • কাজাখস্তান, যুক্তরাষ্ট্র এখন সবচেয়ে বড় মাইনিং হাব

স্মার্ট কন্ট্র্যাক্ট ও DeFi ইকোসিস্টেম

📜 স্মার্ট কন্ট্র্যাক্ট কী?

স্মার্ট কন্ট্র্যাক্ট হলো ব্লকচেইনে থাকা স্বয়ংক্রিয় প্রোগ্রাম যা নির্দিষ্ট শর্ত পূরণ হলে স্বয়ংক্রিয়ভাবে কাজ করে — কোনো মধ্যস্থতাকারী ছাড়া।

উদাহরণ: "যদি A ১ ETH পাঠায়, তাহলে স্বয়ংক্রিয়ভাবে B-কে ১০০ টোকেন দাও" — এটি কোড আকারে ব্লকচেইনে থাকে এবং কেউ থামাতে পারে না।

🔴 Smart Contract Exploit

ব্লকচেইনের সবচেয়ে বড় দুর্বলতা। স্মার্ট কন্ট্র্যাক্টের কোডে বাগ থাকলে হ্যাকার সেটি কাজে লাগায়।

  • Reentrancy Attack: হ্যাকার একটি ফাংশনকে বারবার কল করে — ২০১৬ সালে The DAO থেকে $৬০ মিলিয়ন চুরি হয়েছিল এভাবে
  • Flash Loan Attack: একই লেনদেনে বিশাল পরিমাণ ঋণ নিয়ে বাজার ম্যানিপুলেট করা এবং মুনাফা করা
  • Integer Overflow/Underflow: গাণিতিক সীমা অতিক্রম করে ভুল মান তৈরি করা
  • Access Control Bug: অনুমতি যাচাইয়ের দুর্বলতা কাজে লাগানো

DNS ATTACK

🌐 DNS Hijacking

জনপ্রিয় DeFi ওয়েবসাইটের DNS রেকর্ড পরিবর্তন করে ব্যবহারকারীকে নকল সাইটে নিয়ে যাওয়া। Cream Finance এ এভাবে আক্রমণ হয়েছিল।

ORACLE ATTACK

🔮 Price Oracle Manipulation

DeFi প্রোটোকল বাইরের মূল্য তথ্য (Oracle) ব্যবহার করে। হ্যাকার Flash Loan দিয়ে বাজারে মূল্য ম্যানিপুলেট করে Oracle কে ভুল তথ্য দেয়।

🇧🇩 বাংলাদেশে ক্রিপ্টো — আইনি পরিস্থিতি

বাংলাদেশ ব্যাংক ২০১৪ সালে ক্রিপ্টোকারেন্সি লেনদেনকে নিরুৎসাহিত করেছে এবং সতর্কতা জারি করেছে। তবে ব্লকচেইন প্রযুক্তি নিজে নিষিদ্ধ নয়। ব্লকচেইন ব্যবহার করে সরকারি রেজিস্ট্রি ও সেবা ডিজিটাইজ করার পরিকল্পনা আছে।

বাংলাদেশে ক্রিপ্টো বিনিয়োগ বা লেনদেনের আগে সর্বশেষ বাংলাদেশ ব্যাংকের নির্দেশিকা পড়ুন। আইনি পরিস্থিতি পরিবর্তনশীল।

ব্লকচেইনের ভবিষ্যৎ ও প্রয়োগ

🏥

স্বাস্থ্যসেবা

রোগীর মেডিকেল রেকর্ড ব্লকচেইনে সংরক্ষণ। যেকোনো হাসপাতাল সুরক্ষিতভাবে অ্যাক্সেস করতে পারবে। ওষুধের সাপ্লাই চেইন ট্র্যাক করা।

🗳️

ই-ভোটিং

ব্লকচেইনে ভোট দেওয়া — স্বচ্ছ, যাচাইযোগ্য এবং কারচুপি প্রতিরোধী। এস্তোনিয়া ইতোমধ্যে ডিজিটাল ভোটিং ব্যবহার করছে।

📦

সাপ্লাই চেইন

Walmart, Maersk ব্লকচেইন ব্যবহার করে পণ্য ট্র্যাক করছে। "ফার্ম থেকে টেবিল" — খাবারের উৎস জানা সম্ভব।

🏠

রিয়েল এস্টেট

জমির দলিল ব্লকচেইনে রাখলে জালিয়াতি অসম্ভব। টোকেনাইজেশনের মাধ্যমে ভগ্নাংশে সম্পত্তি কেনা সম্ভব হবে।

🌐

Web3 ইন্টারনেট

কেন্দ্রীয় সংস্থা (Facebook, Google) ছাড়া ইন্টারনেট। ব্যবহারকারীই নিজের ডেটার মালিক। Decentralized Social Media (Lens Protocol, Farcaster)।

💳

CBDC

Central Bank Digital Currency — সরকারি ডিজিটাল মুদ্রা। চীনের ডিজিটাল ইউয়ান চালু। ভারতের ডিজিটাল রুপি পাইলটিং। বাংলাদেশও গবেষণা করছে।

⚡ Layer 2 — ব্লকচেইনের গতি বাড়ানোর সমাধান

Ethereum ধীরে এবং ব্যয়বহুল। Layer 2 সমাধান মূল চেইনের উপরে একটি দ্রুত স্তর তৈরি করে।

  • Lightning Network (Bitcoin): মাইক্রো পেমেন্টের জন্য — কফি কেনা এখন BTC দিয়ে সম্ভব
  • Polygon (Ethereum): গ্যাস ফি ৯৯% কমিয়ে দেয়, সেকেন্ডে হাজারো লেনদেন
  • Arbitrum ও Optimism: Optimistic Rollups প্রযুক্তি ব্যবহার করে
  • zkSync, StarkNet: Zero Knowledge Proof — গণিতের মাধ্যমে যাচাই, কোনো লেনদেনের তথ্য না দিয়ে

⚠️ বিনিয়োগের আগে মনে রাখুন

  • ক্রিপ্টোকারেন্সি অত্যন্ত অস্থির — একদিনে ৫০% নামতে বা উঠতে পারে
  • শুধু সেই অর্থ বিনিয়োগ করুন যা হারালেও আপনার জীবন ভেঙে পড়বে না
  • DYOR (Do Your Own Research) — অন্যের টিপসে অন্ধভাবে বিনিয়োগ করবেন না
  • Diversification — সব টাকা এক কয়েনে রাখবেন না
  • ট্যাক্স সম্পর্কে নিজের দেশের আইন জানুন
