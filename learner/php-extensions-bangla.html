PHP Extension কী?ext_* / .so / .dll

PHP Extension হলো PHP-এর অতিরিক্ত ক্ষমতা যোগ করার পদ্ধতি। PHP কোর খুব ছোট রাখা হয়েছে — শুধু ভাষার মূল কাজগুলো করে। কিন্তু ডাটাবেজ সংযোগ, ছবি প্রক্রিয়াকরণ, ইমেইল পাঠানো, ফাইল কম্প্রেস করা — এসব কাজের জন্য আলাদা Extensions ব্যবহার করতে হয়।

💻 Linux-এ কোথায় থাকে
/usr/lib/php/extensions/ ফোল্ডারে .so ফাইল হিসেবে
🪟 Windows-এ কোথায় থাকে
ext/ ফোল্ডারে .dll ফাইল হিসেবে
⚙️ Enable করার উপায়
php.ini-তে extension=curl লিখে PHP restart করুন
🔍 Check করার উপায়
Terminal-এ php -m বা ব্রাউজারে phpinfo() দেখুন
💡 মনে রাখুন: সব extension enable করা উচিত নয়। শুধু যেগুলো দরকার সেগুলোই চালু রাখুন — এতে PHP দ্রুত চলে এবং সার্ভার নিরাপদ থাকে।

php_curl (cURL)
Client URL Library — HTTP/HTTPS রিকোয়েস্ট পাঠানোর মূল হাতিয়ার
অবশ্যই লাগবে Web APIs

cURL হলো PHP-এর সবচেয়ে গুরুত্বপূর্ণ extension। এটি দিয়ে আপনার PHP কোড থেকে যেকোনো URL-এ HTTP রিকোয়েস্ট পাঠানো যায় — ঠিক যেমন ব্রাউজার করে। REST API কল করা, OAuth login করা, ফাইল ডাউনলোড করা — সব cURL দিয়েই হয়।

🎯 কী কী করা যায়
HTTP GET/POST/PUT/DELETE, ফাইল আপলোড/ডাউনলোড, HTTPS সংযোগ, Cookie পাঠানো, Authorization header সেট করা
📦 Enable করুন
extension=curl (php.ini)
অথবা: apt install php-curl
  • Payment Gateway (SSLCommerz, bKash API, Stripe) সংযোগ করতে
  • Facebook, Google, GitHub OAuth login করতে
  • Weather API, SMS API, Map API ব্যবহার করতে
  • Laravel, Symfony, Guzzle — সব ফ্রেমওয়ার্কই ভেতরে cURL ব্যবহার করে
  • Webhook পাঠাতে ও গ্রহণ করতে
⚠️ cURL ছাড়া কোনো আধুনিক PHP প্রজেক্ট কাজ করে না বললেই চলে। Laravel, WordPress, WooCommerce সবই cURL ব্যবহার করে।
curl_example.php
// cURL দিয়ে API কল করার উদাহরণ
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://api.example.com/users");
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, ["Authorization: Bearer YOUR_TOKEN"]);
$response = curl_exec($ch);
curl_close($ch);
$data = json_decode($response, true);

php_pdo + php_pdo_mysql
PHP Data Objects — ডাটাবেজ সংযোগের সবচেয়ে আধুনিক পদ্ধতি
অবশ্যই লাগবে Best Practice

PDO হলো ডাটাবেজ সংযোগের জন্য একটি unified interface। একই কোডে MySQL, PostgreSQL, SQLite সব ডাটাবেজ ব্যবহার করা যায়। Prepared Statement ব্যবহার করে SQL Injection থেকে সুরক্ষিত থাকা যায়।

✅ সুবিধা
একটাই API দিয়ে যেকোনো ডাটাবেজ, Prepared Statements দিয়ে নিরাপত্তা, ORM লাইব্রেরিগুলো PDO ব্যবহার করে
📦 Enable করুন
extension=pdo_mysql (MySQL-এর জন্য)
extension=pdo_pgsql (PostgreSQL-এর জন্য)
  • Laravel Eloquent, Doctrine ORM — সব PDO ব্যবহার করে
  • MySQL, MariaDB, PostgreSQL, SQLite সংযোগে
  • SQL Injection রোধে Prepared Statement-এর মাধ্যমে

php_gd (GD Library)
সবচেয়ে জনপ্রিয় image processing extension
অবশ্যই লাগবে

GD হলো PHP-এর built-in image library। ছবি তৈরি করা, resize করা, crop করা, watermark দেওয়া, text লেখা, thumbnail বানানো — সব GD দিয়ে করা যায়। JPEG, PNG, GIF, WebP সব ফরম্যাট সাপোর্ট করে।

🎯 কী কী করা যায়
Thumbnail তৈরি, Watermark, Text-to-image, Captcha, Chart/Graph, QR code, ছবির format conversion
📦 Enable করুন
extension=gd
অথবা: apt install php-gd
  • Profile picture upload করলে auto-resize করতে
  • CAPTCHA তৈরিতে (গণিতের ছবি)
  • Certificate বা Invoice PDF-এ ছবি সংযোগে
  • Product image thumbnail বানাতে

php_imagick (ImageMagick)
Professional-grade ইমেজ প্রক্রিয়াকরণ
প্রফেশনাল

  • PDF-এর প্রতিটি পৃষ্ঠা ছবিতে রূপান্তরে
  • PSD, SVG, TIFF ফাইল প্রক্রিয়াকরণে
  • উচ্চমানের ছবি compression ও optimization-এ
  • Batch image processing-এ

php_mbstring (Multibyte String)
বাংলা, আরবি, চীনা — সব ভাষার জন্য অপরিহার্য
অবশ্যই লাগবে বাংলার জন্য জরুরি

PHP-এর সাধারণ string function যেমন strlen(), strtolower() শুধু ASCII character বোঝে। কিন্তু বাংলা, আরবি, হিন্দি বা চীনা ভাষায় কাজ করতে হলে mbstring লাগবে। এটি ছাড়া বাংলা টেক্সট ভেঙে যায়।

🚨 বাংলায় কাজ করলে mbstring অবশ্যই enable করতে হবে। নইলে strlen("বাংলা") ৫ দেওয়ার বদলে ভুল সংখ্যা দেবে কারণ প্রতিটি বাংলা অক্ষর একাধিক byte নেয়।
🔧 mbstring vs normal
Normal: strlen("বাংলা") = ভুল
Mbstring: mb_strlen("বাংলা") = সঠিক ✅
📦 Enable করুন
extension=mbstring
অথবা: apt install php-mbstring
  • বাংলা/আরবি/হিন্দি টেক্সট processing-এ
  • UTF-8 string-এর সঠিক দৈর্ঘ্য, substring, uppercase/lowercase করতে
  • Laravel, Symfony — সব ফ্রেমওয়ার্ক mbstring চায়
  • Email, SMS-এ Unicode বার্তা পাঠাতে

php_intl (Internationalization)
আন্তর্জাতিক ভাষা, তারিখ, মুদ্রা ফরম্যাটিং
Globalization

intl extension দিয়ে বিভিন্ন দেশের নিয়মে তারিখ, সময়, মুদ্রা, সংখ্যা format করা যায়। যেমন বাংলাদেশে ৳১,০০,০০০ কিন্তু USA-তে $100,000 — intl এই কাজ করে।

  • Multi-language website-এ locale-based formatting-এ
  • বিভিন্ন দেশের তারিখ ফরম্যাট (DD/MM/YYYY vs MM/DD/YYYY)-এ
  • মুদ্রা symbol ও দেশ-ভেদে number formatting-এ
  • String comparison যেখানে ভাষার নিয়ম মানা দরকার

php_json
JSON parse ও encode — আধুনিক API-র মূল ভিত্তি
অবশ্যই লাগবে PHP 8-এ Built-in

JSON extension দিয়ে PHP array/object থেকে JSON string বানানো এবং JSON string থেকে PHP array বানানো যায়। REST API, AJAX — সবখানে এটি লাগে।

✅ PHP 8.0+ থেকে JSON built-in হয়ে গেছে, আলাদা enable করতে হয় না। তবে পুরনো PHP-তে extension=json লাগবে।
  • API response তৈরি ও parse করতে: json_encode($data)
  • AJAX থেকে আসা data পড়তে: json_decode($input, true)
  • Config file, settings সংরক্ষণে

  • RSS/Atom feed parse করতে

  • sitemap.xml, configuration XML তৈরিতে
  • HTML parse করতে (DOMDocument)

Cache ও Performance Extensionsopcache · apcu · redis · memcached
🚀
Zend OPcache
PHP-কে ৫-১০ গুণ দ্রুত করে — সবচেয়ে গুরুত্বপূর্ণ performance extension
অবশ্যই Enable করুন Performance

PHP প্রতিটি request-এ .php ফাইল পড়ে, parse করে, bytecode-এ compile করে তারপর execute করে। OPcache এই compiled bytecode মেমোরিতে সংরক্ষণ করে রাখে। পরের request-এ আর compile করতে হয় না — সরাসরি execute হয়।

💡 Production server-এ OPcache সবসময় enable করুন। এটি শুধু একটি লাইন change করলেই PHP ৫-১০ গুণ দ্রুত হয়।
php.ini — OPcache সেটিং
[opcache]
zend_extension=opcache
opcache.enable=1              ; OPcache চালু করুন
opcache.memory_consumption=256    ; 256MB RAM বরাদ্দ
opcache.max_accelerated_files=10000 ; সর্বোচ্চ ফাইল সংখ্যা
opcache.revalidate_freq=60       ; ৬০ সেকেন্ড পর file check করবে
opcache.validate_timestamps=0    ; Production-এ 0 করুন (দ্রুততার জন্য)

php_apcu (APCu)
প্রক্রিয়ার মধ্যে দ্রুত data caching
Recommended In-Memory Cache

APCu হলো PHP-এর user-space memory cache। যেকোনো PHP variable (array, object) RAM-এ সংরক্ষণ করা যায় এবং পরে আবার পড়া যায়। Database query result, config data — এসব cache করতে ব্যবহার করুন।

  • ঘন ঘন ব্যবহৃত DB query result cache করতে
  • Session-এর বিকল্প হিসেবে
  • Rate limiting counter রাখতে
  • Small website-এ Redis ছাড়াই caching করতে

php_redis / php_memcached
External cache server সংযোগ
Production Cache

Redis ও Memcached হলো আলাদা cache server। PHP extension দিয়ে এই server-এর সাথে কথা বলা যায়। Laravel-এর Cache, Queue, Session সব Redis-এ রাখা যায়।

Redis এর সুবিধা
Data structure support (List, Set, Hash), Persistence, Pub/Sub, Laravel Queue-এ ব্যবহার
Memcached এর সুবিধা
একদম সরল key-value store, multi-server support, খুব দ্রুত

অবশ্যই লাগবে

OpenSSL extension ছাড়া HTTPS ব্যবহার করে কোনো URL-এ request পাঠানো যায় না। cURL এর HTTPS, JWT token, SSL certificate — সব OpenSSL-এর উপর নির্ভর করে।

  • HTTPS API call করতে (cURL-এর সাথে)
  • JWT (JSON Web Token) তৈরি ও verify করতে
  • Email encryption (S/MIME)-এ
  • RSA/AES encryption/decryption করতে
  • SSL Certificate পড়তে ও যাচাই করতে
🚨 OpenSSL ছাড়া Payment Gateway-র HTTPS API call করা সম্ভব নয়। এটি সবসময় enable রাখুন।
🧂
php_sodium (libsodium)
আধুনিক cryptography — সবচেয়ে নিরাপদ encryption
PHP 7.2+ Built-in Modern Crypto

Sodium হলো OpenSSL-এর চেয়ে আধুনিক ও নিরাপদ cryptography library। PHP 7.2 থেকে built-in। Password hashing, secret key encryption, digital signatures সব করা যায়।

  • API key encryption করতে
  • End-to-end encrypted message তৈরিতে
  • Secure random number তৈরিতে
#️⃣
php_hash
MD5, SHA256, bcrypt — হ্যাশিং ফাংশন
PHP-তে Built-in

Hash extension দিয়ে MD5, SHA-1, SHA-256, HMAC সব ধরনের hash তৈরি করা যায়। Password hash করতে password_hash() ব্যবহার করুন (bcrypt ব্যবহার করে)।

⚠️ Password store করতে কখনো MD5 বা SHA1 ব্যবহার করবেন না। সবসময় password_hash($pass, PASSWORD_BCRYPT) ব্যবহার করুন।

php_zip
ZIP ফাইল তৈরি ও খোলা
Recommended

ZIP extension দিয়ে PHP থেকে ZIP ফাইল তৈরি করা, extract করা যায়। Composer PHP package ইনস্টল করতে এটি ব্যবহার করে।

  • Backup system-এ ফাইল compress করতে
  • Multiple ফাইল একসাথে download করাতে
  • Composer package management-এ (অবশ্যই লাগে)
  • Laravel zip package তৈরিতে
🔍
php_fileinfo
ফাইলের আসল ধরন detect করা
Security

Fileinfo দিয়ে ফাইলের extension দেখে নয়, ভেতরের content দেখে আসল file type বের করা যায়। কেউ malware.php কে malware.jpg নাম দিয়ে upload করলেও fileinfo ধরে ফেলে।

🛡️ File upload security-র জন্য সবসময় finfo_file() দিয়ে MIME type যাচাই করুন। শুধু extension দেখা নিরাপদ নয়।

php_zlib
gzip compression — bandwidth সাশ্রয়
Generally Built-in

zlib দিয়ে data compress ও decompress করা যায়। HTTP response gzip করে পাঠালে bandwidth ৭০% পর্যন্ত কমে।

  • HTTP output compression (Apache/Nginx-এ)
  • Database-এ বড় data compress করে store করতে
  • .gz ফাইল পড়তে ও লিখতে

php.ini — সম্পূর্ণ বাংলা গাইডConfiguration File ব্যাখ্যা

php.ini হলো PHP-এর প্রধান configuration ফাইল। এখানে সব Extension enable/disable করা হয়, memory limit, upload size, error reporting, timezone — সব সেটিং এখানে থাকে। ফাইলটি সাধারণত /etc/php/8.x/apache2/php.ini বা /etc/php/8.x/fpm/php.ini-তে পাওয়া যায়।

php.ini — সম্পূর্ণ ব্যাখ্যাসহ উদাহরণ
;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;
;; PHP.INI — বাংলায় বিস্তারিত ব্যাখ্যা
;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;

;; === EXTENSION লোড করার নিয়ম ===
;; extension= দিয়ে সাধারণ extension লোড হয়
;; zend_extension= দিয়ে Zend engine extension (OPcache) লোড হয়

[PHP]

; ====== BASIC ENGINE SETTINGS ======

engine = On             ; PHP engine চালু রাখুন (সবসময় On)
short_open_tag = Off     ; <? এর পরিবর্তে <?php ব্যবহার করুন (Off রাখা ভালো)
precision = 14          ; দশমিক সংখ্যায় কতটি অঙ্ক দেখাবে

; ====== MEMORY ও EXECUTION ======

memory_limit = 256M     ; একটি PHP script সর্বোচ্চ কতটুকু RAM ব্যবহার করতে পারবে
                             ; Laravel: ন্যূনতম 128M, ভিডিও/ছবি হলে 512M বা বেশি

max_execution_time = 30  ; script সর্বোচ্চ ৩০ সেকেন্ড চলবে, তারপর বন্ধ হবে
                             ; বড় file import বা batch job-এ বাড়াতে হতে পারে (300)

max_input_time = 60     ; POST/GET data গ্রহণ করতে সর্বোচ্চ ৬০ সেকেন্ড

; ====== FILE UPLOAD সেটিং ======

file_uploads = On       ; ফাইল আপলোড চালু রাখুন

upload_max_filesize = 64M ; সর্বোচ্চ আপলোড ফাইল সাইজ (64MB)
                             ; এই সীমা বাড়ালে post_max_size-ও বাড়াতে হবে

post_max_size = 128M    ; একটি POST request-এ সর্বোচ্চ কতটুকু data পাঠানো যাবে
                             ; upload_max_filesize-এর চেয়ে বড় রাখতে হবে

upload_tmp_dir = /tmp   ; আপলোড করা ফাইল সাময়িকভাবে কোথায় রাখবে

max_file_uploads = 20   ; একসাথে সর্বোচ্চ কতটি ফাইল আপলোড করা যাবে

; ====== ERROR REPORTING ======

display_errors = Off     ; Production-এ Off রাখুন (error browser-এ দেখাবে না)
                             ; Development-এ On করুন

display_startup_errors = Off ; PHP startup error দেখানো হবে কিনা

log_errors = On          ; Error log ফাইলে লিখুন (সবসময় On রাখুন)

error_log = /var/log/php/error.log  ; Error কোথায় লগ হবে

error_reporting = E_ALL  ; কোন ধরনের error report করবে
                             ; E_ALL = সব error | Production: E_ALL & ~E_DEPRECATED

; ====== DATE/TIME ======

date.timezone = Asia/Dhaka  ; বাংলাদেশের timezone সেট করুন
                                ; না করলে PHP warning দেবে

; ====== SESSION সেটিং ======

session.save_handler = files    ; session কোথায় সংরক্ষণ হবে (files / redis / memcached)
session.save_path = "/var/lib/php/sessions"
session.gc_maxlifetime = 1440  ; session কতক্ষণ বৈধ থাকবে (সেকেন্ডে = ২৪ মিনিট)
session.cookie_secure = 1     ; HTTPS-এ cookie পাঠাও (Production-এ 1 করুন)
session.cookie_httponly = 1  ; JavaScript থেকে cookie access করা বন্ধ (XSS রোধে)
session.cookie_samesite = Strict ; CSRF attack রোধে

; ====== EXTENSIONS ENABLE করা ======

extension=curl          ; HTTP request পাঠাতে (অবশ্যই চালু করুন)
extension=mysqli        ; MySQL সংযোগ (WordPress ব্যবহার করে)
extension=pdo_mysql     ; PDO দিয়ে MySQL (Laravel, Symfony)
extension=pdo_pgsql     ; PDO দিয়ে PostgreSQL
extension=pdo_sqlite    ; SQLite ডাটাবেজ
extension=gd            ; ছবি resize, thumbnail, watermark
extension=mbstring      ; বাংলা/আরবি ভাষার জন্য অবশ্যই লাগবে
extension=openssl       ; HTTPS, SSL, JWT, encryption
extension=json          ; JSON parse/encode (PHP 8-এ built-in)
extension=xml           ; XML parse করতে
extension=simplexml     ; সহজ XML parsing
extension=dom           ; DOM XML manipulation
extension=zip           ; ZIP ফাইল তৈরি ও extract
extension=fileinfo      ; ফাইলের আসল type detect করা
extension=intl          ; Internationalization (বহু-ভাষা)
extension=bcmath        ; সঠিক বড় সংখ্যার গণনা (টাকার হিসাব)
extension=sodium        ; আধুনিক cryptography
extension=redis         ; Redis cache server সংযোগ
extension=apcu          ; In-memory PHP caching
extension=imap          ; Email server থেকে mail পড়া
extension=soap          ; SOAP Web Service
extension=sockets       ; TCP/UDP socket connection
extension=tokenizer     ; PHP code parsing (Composer লাগে)
extension=exif          ; ছবির metadata (GPS, camera model)

; Zend Extension (OPcache) — সবার নিচে থাকবে
zend_extension=opcache  ; PHP-কে ৫-১০x দ্রুত করে — সবসময় চালু রাখুন!

; ====== OPCACHE সেটিং ======
[opcache]
opcache.enable=1
opcache.memory_consumption=256        ; cache-র জন্য RAM (MB)
opcache.interned_strings_buffer=16   ; string buffer (MB)
opcache.max_accelerated_files=10000  ; কতটি ফাইল cache করবে
opcache.validate_timestamps=1       ; Development-এ 1, Production-এ 0
opcache.revalidate_freq=0           ; কতক্ষণ পর file check করবে (0=প্রতি request-এ)
php.ini directive Development Production কী করে / কেন
display_errors On Off error browser-এ দেখানো হবে কিনা। Production-এ Off করুন নইলে হ্যাকার সুবিধা পাবে
error_reporting E_ALL E_ALL & ~E_DEPRECATED কোন ধরনের PHP error report করবে
memory_limit 512M 256M script-এর সর্বোচ্চ RAM ব্যবহার
max_execution_time 120 30 script কতক্ষণ চলতে পারবে (সেকেন্ড)
upload_max_filesize 100M 32M সর্বোচ্চ upload ফাইল সাইজ
post_max_size 128M 64M একটি POST request-এ সর্বোচ্চ data। upload size-এর চেয়ে বড় রাখুন
date.timezone Asia/Dhaka Asia/Dhaka বাংলাদেশ timezone। না দিলে PHP warning দেবে
session.cookie_secure 0 1 HTTPS-এ session cookie পাঠাবে। HTTP-তে পাঠাবে না
session.cookie_httponly 1 1 JavaScript থেকে session cookie পড়া বন্ধ — XSS রোধে
opcache.enable 1 1 PHP দ্রুততার জন্য সবসময় চালু রাখুন
opcache.validate_timestamps 1 0 Development-এ 1 (file change দেখবে), Production-এ 0 (দ্রুততার জন্য)
expose_php On Off HTTP header-এ PHP version দেখানো। Production-এ Off করুন — security
allow_url_fopen On Off URL থেকে file_get_contents() করতে পারবে কিনা। cURL ব্যবহার করুন পরিবর্তে
allow_url_include Off Off Remote URL থেকে PHP include করা বন্ধ — সবসময় Off রাখুন (security)
🔄 php.ini পরিবর্তন করার পরে Apache: sudo systemctl restart apache2 এবং PHP-FPM: sudo systemctl restart php8.x-fpm দিয়ে restart করতে হবে।
🔍 php.ini-র location জানতে terminal-এ php --ini দিন। অথবা PHP ফাইলে <?php phpinfo(); ?> লিখে browser-এ open করুন — সব তথ্য দেখাবে।

Payment Gateway / API call করতে cURL + OpenSSL extension=curl
MySQL ডাটাবেজ ব্যবহার করতে PDO + pdo_mysql extension=pdo_mysql
বাংলায় text processing করতে Mbstring extension=mbstring
ছবি resize/thumbnail করতে GD বা Imagick extension=gd
PHP দ্রুত করতে OPcache zend_extension=opcache
Cache রাখতে (Redis) php_redis extension=redis
ZIP ফাইল বানাতে php_zip extension=zip
Password সুরক্ষিত রাখতে Hash (built-in) আলাদা extension লাগে না
Composer ইনস্টল করতে zip + tokenizer + mbstring তিনটিই enable করুন
Laravel চালাতে curl + pdo_mysql + mbstring + openssl + zip + gd + json + xml সবগুলো enable করুন
WordPress চালাতে mysqli + gd + mbstring + curl + zip এই পাঁচটি অবশ্যই
